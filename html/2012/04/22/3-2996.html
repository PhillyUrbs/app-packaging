In this installer, I first manually installed and configured NOD32 on a client machine, then used the import/export tool to create the XML configuration file like so:


right click the NOD32 icon in the system tray


select "Open Window"


Click "Setup"


Click the "Import and export settings..." link


Select "Export settings" and provide a file name (use "cfg.xml" unless you know what you are doing)


You can now use msiexec.exe with the parameters given above to deploy NOD32v3 and set it up using the xml configuration file.


In my case, I was installing NOD32 through the KBox.  If you are using the KBox also, keep reading.


After exporting the configuration file, create a zip file containing cfg.xml, the NOD32 package (eavb_nt32_enu.msi) and msiexec.exe (might not be necessary, but I didn't figure out how to do it without including msiexec.exe).  Create a distribution in the KBox using the zip file and the msiexec.exe parameters given above.  That's it!


Edit: 01/13/2012


The install notes above refer to the steps I took to install NOD32 on Windows XP, Windows 2000 Server and Windows 2003 Server.  I am currently trying to get NOD32 to install from the KBox on Windows 2008 server, and am running into issues.  If you are trying to install on Win2K8 Server, be advised the steps above may not work for you.
