Setup.exe supports silent installation when it is invoked from a command line with the InstallShield -r and -s switches. Setup.iss is a response file containing a script that is automatically generated by the SETUP program when you use the -r switch. With the Setup.iss file, the -s switch will initiate a silent installation of TSM. Page 65 of the quck start guide (see link below) covers these steps which are the same as you would perform for any other InstallShield setup.


Source: Tivoli Storage Manager For Windows NT: Quick Start
