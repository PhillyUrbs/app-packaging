If you want to use a 32 Bit Browser under 64 Bit Windows 7 you need the 32 Bit Java Runtime environment, too.


I got a MSI Error "1619" while installing the 32 Bit JRE with the System Account (in SCCM). The Problem is the wrong Path to the MSI File.


So I installed with the MSI and it works. To get the MSI File you have to start the exe File and then go to:


C:\Users\%username%\AppData\LocalLow\Sun\Java\jre1.6.0_2x


Then you can use the Msi File:


jre1.6.0_2x.msi /quiet /norestart ADDLOCAL=ALL IEXPLORER=1 MOZILLA=1 REBOOT=Suppress JAVAUPDATE=0
