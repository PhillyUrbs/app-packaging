A standard default install can be done silently with the following command line:

%windir%\system32\msiexec /i "Adobe Illustrator CS2.msi" /qb+ SERIALNUMBER="1034-XXXX-XXXX-XXXX-XXXX-XXXX"

For our company, I also created a transform moving the shortcuts to a folder nemed Adobe. I also dropped in some registry entries to get rid of the registration nag screen at first launch:

REGEDIT4

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\{B2F5D08C-7E79-4FCD-AAF4-57AD35FF0601}]
"EPIC_REGS_COUNT"=dword:00000000
"EPIC_REGS_DATE"="2005-06-21 10:37"
"EPIC_REGS_LANG"=dword:00000000
"EPIC_REGS_STATE"=dword:00000002
"EPIC_REGS_TYPE"=dword:00000004

I found these keys also work for Photoshop CS2 and the Bridge componenet. You need to change the GUID above to match the product, of course.

If any of the additional components are required, such as the help center or stock photography, you need to create a script file or BAT file to run the MSI's in sucessive order starting with the main Illustrator MSI.

Between the command line and registry entries, I have been able to silently install sucessfuly on W2K and WXP.

For me, the above information worked.  However, I did not have to add any additional registry entries to get rid of the "Registration" dialog.

These keys did the trick:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\{B2F5D08C-7E79-4FCD-AAF4-57AD35FF0601}]
"EPIC_REGS_TYPE"=dword:00000004
"EPIC_REGS_STATE"=dword:00000002
"EPIC_REGS_LANG"=dword:00000000
"EPIC_REGS_COUNT"=dword:00000000
"EPIC_REGS_DATE"="2006-01-09 14:56"

Also, to get rid of the "Updates" dialog, I simply editted the
"C:\Documents and Settings\All Users\Application Data\Adobe\Updater\AdobeESDGlobalApps.xml"

I changed all the "<IsEnabled>1</IsEnabled>" entries to "<IsEnabled>0</IsEnabled>".

Note: It is easier to edit this file with Wordpad as opposed to Notepad.  Simply make this modification in your MST and you are all set.

From the adobe website, to disable the updater service, which will disable the update service for all adobe products add the following registry key:

[HKEY_LOCAL_MACHINE\SOFTWARE\Adobe\Updater]
"Enterprise"=dword:00000001 "

If the "Enterprise" key is missing or has any other values, updates are enabled.

http://www.adobe.com/support/techdocs/331841.html
