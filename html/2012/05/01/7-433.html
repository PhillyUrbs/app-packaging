The 30mb psp704ev.exe setup file is a self extracting archive which places its files at, "C:\Program Files\Jasc Software Inc\Setup Files\Paint Shop Pro 7 Try and Buy" by default.

Using these source files, Paint Shop Pro may be installed using standard MSI switches, for example:

msiexec /i "C:\Program Files\Jasc Software Inc\Setup Files\Paint Shop Pro 7 Try and Buy\Paint Shop Pro 7 Try And Buy.msi" ALLUSERS=2 /qb

My goal: distribution using Active Directory.

I've had 2 problems with this program:
1. Unattend uninstall stops
2. Sofware asks for registration and file associations

Solved this by creating an administrative install.
Then I made a tranform using Wise Package Studio 5

Disabled all the custom actions named "EXE_*_JUNREG"

Created a new component "CurrentUser" and imported the following REG-settings:

-------------------------------------------------
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Jasc\ereg]
"RegState"="Fail"

[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 7\General]
"Version"=dword:07000204
"MRUSolidColors"="24BFF80020E2"
"ERegAttempted"=dword:00000001
"AreBarsCustomized"=dword:00000000

[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 7\Tip]
"TimeStamp"="Wed Feb 14 07:02:00 2001"
"StartUp"=dword:00000000
"FilePos"=dword:000000d1

----------------------------------------------

Greetz, Wouter Heesakkers (NL)

Seems it will install and run without a serial number.

"%DP0Paint Shop Pro 7.msi" ALLUSERS=1 ROOTDRIVE=C:\ REBOOT=RS /L*v "%TEMP%\PAINT_SHOP_PRO_7_04.log" /qb

If you care to enter it at install time:
ISX_SERIALNUM=<serial number>


Using Active setup, did the following for each user:

Windows Registry Editor Version 5.00

;To Hide the registration window at first startup of paintshop pro
[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 7\General]
"ERegAttempted"=dword:00000001

;To Hide the registration window at first startup of Animation Shop 3
[HKEY_CURRENT_USER\Software\Jasc\Animation Shop 3\General]
"ERegAttempted"=dword:00000001

;To Hide the tips at startup of paintshop pro
[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 7\Tip]
"StartUp"=dword:00000001

The psp704ep.exe patch will not install unattended or silently. If you need to make it so, capture the patch's installation and create a Transform for the original MSI.

I've packaged this in a similar fashion (Active Setup) with the following registry entries:-

[HKEY_CURRENT_USER\Software\Jasc\Animation Shop 3\General]
"ERegAttempted"=dword:00000001
"Version"=dword:03000204

[HKEY_CURRENT_USER\Software\Jasc\ereg]
"RegState"="Fail"

[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 7\General]
"ERegAttempted"=dword:00000001
"Version"=dword:07000204

[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 7\Tip]
"StartUp"=dword:00000001

Please note, that if you want a silent uninstall of the product, then remove/comment the EXE_ENGLISH_JUNREG Custom Action.
