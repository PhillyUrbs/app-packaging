Installation of SAP BPC Client is posible by using this command line:



<installername> /s /z"<install-dir>&<Xceedzip.dll

path>&<language-selection>&<feature-selection>"



Example:



"BPCCLIENT10_0-10009079.EXE" /s /z"C:\Program Files\SAP BusinessObjects\PC_NW&c:\Temp\xceedzip.dll&NL&ALL"



For some reason the language package did not catch. Therefor i created a extra package to set the registry to the following:



REGEDIT4



[HKEY_LOCAL_MACHINE\SOFTWARE\OUTLOOKSOFT]

"+"=""



[HKEY_LOCAL_MACHINE\SOFTWARE\OUTLOOKSOFT\50]

"+"=""



[HKEY_LOCAL_MACHINE\SOFTWARE\OUTLOOKSOFT\50\Admin]

"Language"="NL"

"PRGPATH"="C:\\Program Files\\SAP BusinessObjects\\PC_NW\\"

"SETUPTYPE"="SMS"

"TYPE"="FULL"

"UPDATEVERSION"="7.50.10"

"VERSION"="7.50.10"



[HKEY_LOCAL_MACHINE\SOFTWARE\OUTLOOKSOFT\50\CLIENT]

"ALIVEPERIOD"="1800000"

"DATA_PATH"="C:\\Temp"

"defaultlanguage"="NL"

"PATCHVER"="0000"

"PRGPATH"="C:\\Program Files\\SAP BusinessObjects\\PC_NW\\"

"SETUPTYPE"="SMS"

"supportlanguage"="EN,NL"

"TYPE"="FULL"

"USER"="Company"

"VERSION"="7.50.10"

Removing the installation silent:

<installername> /s /x /z"Uninstall"

Example:

"BPCCLIENT04_0-10009079.exe" /s /x /z"uninstall"
