Microsoft recommends for SMS 2003 client removal (when not upgrading to SCCM) that you use ccmclean.exe. Switches I've used are as follows:

ccmclean.exe /keephistory /q

/? - Display Help
/client - Remove the SMS Advanced Client only (default)
/mp - Remove SMS Management Point only
/logdir:<dir> - Set the location where log files should be created
/logbackup:<dir> Backs up SMS Advanced client andMP log files
/keephistory - Keeps software distribution and inventory history when removing
/q - quiet
