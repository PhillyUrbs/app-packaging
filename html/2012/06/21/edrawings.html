With a simple command line you can deploy edrawings by script or by sms. I used the following command line:

msiexec /i "eDrawings.msi"  ADDLOCAL=All  /qn

That command inserted in SMS  can install edrawings in silent mode. You can use also some script, for example in the login, in this case you must launch the command line as an administrator, for example:

RUNAS /user:yourdomainname\administrator yourscript.bat
