[HKEY_CURRENT_USER\Software\Sprint\Sprint PCS Connection Manager]
"UpdateAutoCheck"=dword:00000000

Some changes with registry paths/settings in v3.07.037;
"Start in system tray" turned on;
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband]
"Minimize"=dword:00000001
"Always on top" turned off:
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband]
"Topmost"=dword:00000000
"Play sound effects" turned on:
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband]
"EnableSound"=dword:00000001
"Check for updates once per month" turned off:
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband]
"UpdateAutoCheck"=dword:00000000
"Launch when Windows starts" turned off:
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband]
"StartupLaunch"=dword:00000000
"Always show Welcome Screens" turned on:
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband\Modules\CDMA Access]
"EnableWelcome"=dword:00000001
"GO automatically when ready" turned off:
[HKEY_CURRENT_USER\Software\Sprint\Sprint Mobile Broadband\Modules\CDMA Access]
"EnableGoWhenReady"=dword:00000000
