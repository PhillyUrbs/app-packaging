Silent installation without userinteraction:
(You have to extract the exe file before)
setup.exe /install /quiet=31 /nodisp /reboot=0
I just added the serial number with a reg file afterwards:
Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SOFTWARE\FinePrint Software\pdfFactory4]
"Version"=dword:00000432
"RegName"="Registrationname"
"RegNum"="XXXXX-XXXXX-XXXXX"

Run setup.exe with the following Command line:

    /install  -    installs SE

    /uninstall - removes SE

    /quiet=Z    ; Z can be a combination of the following values:

    1 = suppress the opening confirmation dialog
    2 = suppress the progress bar
    4 = suppress the closing message box
    8 = suppress the license agreement
    16 = suppress the “must reboot” message box (default to Yes)

    For example, to suppress the progress bar and the closing message box, you must specify /quiet=6 (2 for the progress bar plus 4 for the   
    closing message box).

      /quiet=31 -  suppresses ALL output during installation or uninstallation (silent install)

    /autodelete(installation only) -  tells the installer to delete the entire source directory after installation has completed

    /inf  -              does not actually install or uninstall; generates INF files for Add Printer Wizard setup on the current platform

    /reboot=Z      controls the reboot initiated by the installer or uninstaller. Z can be one of the following values:
                            0 suppresses a reboot. This does not prevent the need for rebooting the machine; it simply tells the installer/uninstaller not to 
                            prompt the user to reboot.

                            1 forces a reboot. If the 16 value is specified in the /quiet option, then the machine will be rebooted without any prior warning; 
                              otherwise the user will be prompted to reboot.
