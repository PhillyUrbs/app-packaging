The following installer behaviors can be customized in the Property table:


TSC_SOFTWARE_KEY

The registered software key.


TSC_EVALEMAIL

Show the Tips and Tricks e-mail signup window.


TSC_EMAIL_SIGNUP

Sign up for SnagIt Tips and Tricks.


TSC_START_AUTO

Run SnagIt when Windows starts.


START_NOW

Run SnagIt when the installation finishes.


TSC_DESKTOP_LINK

Create a shortcut to SnagIt on the desktop.

A couple of things that I found useful when creating a company wide deployment.


To Disable Registration Dialog:

Add the following reg key.

HKEY_LOCAL_MACHINE\SOFTWARE\TechSmith\SnagIt\9

reg_dword NoRegistrationDialogSupport 1


To Disable Auto Update function:

Add the following reg key.

HKEY_LOCAL_MACHINE\SOFTWARE\TechSmith\SnagIt\9

reg_dword NoAutoUpdateSupport 1


To Disable the snagit printer installation:

Add the following reg key.

HKEY_LOCAL_MACHINE\SOFTWARE\TechSmith\SnagIt\9

reg_dword NoPrinterInstallationSupport 1


To Turn the customer improvement option to 'No Thanks'

Add the following reg key.

HKEY_CURRENT_USER\Software\TechSmith\SnagIt\9

reg_dword TUDIOptedIn 2

SnagIt9.msi TSC_DESKTOP_LINK="" /log c:\msi.log /passive


Or you have to remove the property TSC_DESKTOP_LINK from the property table of the msi or mst.

There are some inconsistencies in Techsmith's documentation. Their deployment guide is written for "Version 8 and higher". The registry key they specify that controls whether or not the application checks for updates automatically does not exist in version 9 (they specify HKEY_CURRENT_USER\Software\Techsmith\Snagit\9\NoAutoUpdateSupport). The actual registry key for this is:

HKEY_CURRENT_USER\Software\TechSmith\SnagIt\9\Updater

This key has the values:

PromptToCheck

AllowDateBasedCheck

LastCheck

DaysBetween


AllowDateBasedCheck turns on (dword 1) and off (dword 0) automatic updating

PromptToCheck changes whether or not it prompts if there is an update

DaysBetween determines how often to check (every X days)

When uninstalling Snagit v9 you're presented with a screen to save or delete your data files....to disable this feature when silently uninstalling use this command:


:: Uninstall Snagit v9 silently....and hide the nag screen.


MsiExec.exe /X{Product ID} TSC_DATA_STORE=0 /quiet /norestart
