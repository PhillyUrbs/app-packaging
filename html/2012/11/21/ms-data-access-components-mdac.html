Dasetup.exe is the setup engine that is used for MDAC 2.5 and MDAC 2.6. Dasetup.exe performs all the dependency checks and the installation of files, and also sets the registry values. Depending on the status of the installation, Dasetup.exe returns one of the values specified below...


0 (ERROR_SUCCESS)

Setup completed successfully.


3010 (ERROR_SUCCESS_REBOOT_REQUIRED)

Setup completed successfully, but you must restart the computer for MDAC to be installed completely.


0x8000 04005 (E_FAIL)

Setup failed to complete successfully.


1223 (ERROR_CANCELLED)

The user cancelled Setup.


112 (ERROR_DISK_FULL)

There is not enough disk space to complete Setup.


Dasetup.exe is installed in the following folder:

System Drive:\Program Files\Common Files\Microsoft Shared\Dasetup


Source: MDAC 2.6 Setup FAQ
