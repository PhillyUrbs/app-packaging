Dear Packager ;)


I've created an Package of "IBM iSeries Access for Windows" Version 6 Release 1 (V6R1M0)

I copy only DVD 1, there is all for 32- and 64-bit on it. Under "...\Windows\image32\" and "...\Windows\image64a\" you can delete the MRIxxx folder (language folders), that aren't necessary for your installation. So you can resize the package from around 3500MB to 600MB, when you only use German and English as example.


Start Installation with setup.exe silent and using msi Parameter:


"...\Windows\image32\setup.exe" /S /v"REBOOT=REALLYSUPPRESS ADDLOCAL=inavbo,emu,scs,afp,odbc,dt,dtexcel /qn /Li \"%temp%\IBM_CA_610_Install32.log\""


With the /v switch for setup.exe (Installshield) you can control the msi Switches.


To install, upgrade, or add features, enter this command: setup /vADDLOCAL=featurename, where featurename is the internal name of the feature.


To include more than one feature, separate the features with a comma as shown in this example: setup /vADDLOCAL=dir,irc,emu


To install, upgrade, or add all features, enter this command: setup /vADDLOCAL=ALL


To remove features after the installation or upgrade, enter this command: setup /vREMOVE=featurename, where featurename is the internal name of the feature.


To remove the product, enter this command: setup /vREMOVE=ALL
