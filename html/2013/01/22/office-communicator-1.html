Note
Create a standard response transform. Download the latest Cumulative Update (at time of writing KB978564 April 2010).

You can then install Communicator and patch it with one command line:

start /w msiexec /i communicator.msi /update communicator.msp TRANSFORMS=communicator.mst ALLUSERS=2 REBOOT=ReallySuppress /qn /Liv %temp%\Communicator.log

After install, we merge in a .reg key with the following:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Communicator\SessionManager\Apps\{27877e66-615c-4582-ab88-0cb2ca05d951}]
"Name"="Share Information Using Office Communications Server"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run]
"Communicator"=-


The first reg entry changes the "Share Information Using Live Meeting" text in the app to "Share Information Using Office Communications Server". The second one prevents Communicator from launching on Windows statup .. useful if you have a reboots to do as you're setting up your machine prior to imaging it. When a user manually launches it, it should launch for them on subsequent logins.

Command Line
If you are using the Communicator 2007 R2 MUI package for additional language support, DEFAULTLANGUAGE=%CODE% can be used for setting default language during installation.

I found this useful when installing the Swedish version of
Communicator 2007 R2 and then adding the MUI package, since it kept defaulting to English after the MUI had been installed.
