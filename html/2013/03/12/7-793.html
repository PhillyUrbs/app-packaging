Every configuration option available for this client is located in an "INOC6.icf" file in the root of the installation source directory.  You can copy and change that files name to whatever you want, but if you run setup.exe /s, it requires that file to be named "INOC6.ICF".  If you want to change it, you must run MSIEXEC /I "Path to MSI" INO_ICFFILE="Path to ICF" /qb. 


Really, all they did was make a few enhancements to InoculateIT 6.0 and rebranded it as eTrust Antivirus 7.1.


The second file you will need to edit is "INODIST.INI".  This file contains the location to your Distribution servers.  Below is a sample of my recommended config.

---------------

[SOURCES]

1=UNC_0

2=FTP_0


[UNC_0]

Method=UNC

Path=\\YOURSERVER\inoupd$

UserName=

UserPassword=

RedistGui=1


[FTP_0]

Method=FTP

HostName=ftpav.ca.com

UserName=anonymous

UserPassword=Somebody@somecompany.com

FastConnection=0

ProxyName=

UpdatePath=/pub/inoculan/scaneng/

---------------

This works great, because in the event that your distro server crashes, you have the FTP as a backup method.


This installation does a great job working as an upgrade or new install.  I have more details for anyone else interested.
