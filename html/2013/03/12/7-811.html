Deployment Tip Summary

Oracle Java 7 Update 2 (64bit)

Install with .exe:

jre-7u2-windows-x64.exe /s

Install with .exe and additional Parameters:

jre-7u2-windows-x64.exe /s /v"/norestart AUTOUPDATECHECK=0 JAVAUPDATE=0 JU=0"

Paramters:

AUTOUPDATECHECK = enables or disables automatic check for update

JAVAUPDATE = enables or disables updates

JU = enables or disables updates

MOZILLA = 1 = register java with mozilla

IEXPLORER = 1 = register java with ie
