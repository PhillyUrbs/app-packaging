Note

I used WPS to generate the .msi.  After compile, there were two things to clean up:


1.  Several zero byte files had entries in the files table.  Deleting them cleared the compile errors.


2.  When testing the install from the WPS generated .msi, Windows Installer reported that the folder path contained invalid characters.  For some reason, some " marks were added to some DefaultDir path values in the Directory table.  Removing them and recompiling resolved the issue.
