Hello All,

To Automatically accept EULA for PSP 9.x, please set the following reg key
HKCU\software\JASC\Paint Shop Pro 9\General\EULA=1 (DWORD value)

To Not to see the ERegistration window at the startup of PSP, please set the following reg key
HKCU\software\JASC\Paint Shop Pro 9\General\ERegAttempted=1(DWORD Value)

117
Note
To prevent the display of File Format Association dialog box, set the following registry keys:

[HKEY_CURRENT_USER\Software\Jasc\Paint Shop Pro 9\General]
"RunOnce"=dword:00000001 (1)

To set the Automatic check for update interval to Never check for update in Jasc update dalog set the registry key

[HKEY_CURRENT_USER\Software\Jasc\Jasc Update\{F843C6A3-224D-4615-94F8-3C461BD9AEA0}]
"Interval"=dword:00000003 (3)
"LastCheckDay"=dword (dd)
"LastCheckMonth"=dword (mm)
"LastCheckYear"=dword: (yyyy)

Where dd-mm and yyyy correspond to any date before the current date.

To set the default automatic check for update interval to other values set the following keys

Daily "Interval"=dword:00000001 (0)
Weekly "Interval"=dword:00000001 (1)
Monthly "Interval"=dword:00000002 (2)
