Note

Download the MSI for the pre-deployment. (anyconnect-win-2.5.3055-pre-deploy-k9,msi)  You can then deploy silently thru your preferred EDS.  

Create a Profile.xml to be used to configure Client settings and Connections.

Copy the Profile.xml to the following locations after install. 
C:\ProgramData\Cisco\Cisco AnyConnect VPN\Profile - Windows Vista/7
C:\Documents and Settings\All Users\Application Data\Cisco\Cisco AnyConnect VPN Client\Profile 
(I created a seperate MSI to perform this action and made it a dependency for the AnyConnect Client install thru SCCM 2012) 

If you are already using a profile for Web based deployments, you can use the existing profile XML by exporting it from the ASDM console.

Simple Profile XML's can be created using the  AnyConnect Profile Editor.  This is also available for download.
http://www.cisco.com/cisco/software/release.html?mdfid=282414594&softwareid=282364364&release=Profile%20Editor
