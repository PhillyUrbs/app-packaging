Note

Macromedia offers a full installer download for Flash and Shockwave. Admins who need the full install should complete a license agreement at:

http://www.macromedia.com/support/shockwave/info/licensing/main_3.html


The URL below downloads Flash_Shockwave_Full.exe.

http://www.macromedia.com/go/full_shockwaveplayer_win


Running "Flash_Shockwave_Full.exe /S" initiates a silent install with for Flash 6 and Shockwave 8.5.


Extracting Flash_Shockwave_Full.exe with PowerArchiver reveals:


INSTALL FLASH PLAYER 6 AX.EXE

INSTALL FLASH PLAYER 6.EXE

SHOCKWAVE_INSTALLER_FULL.EXE


Should you need to install just Flash Player 6 (Active X version), Flash Player 6, or Shockwave; you can run the extracted installers individually. Though not documenation seems to be available, I found that the following switches work:


Silent install for Flash Player AX (Flash Player Active X):


run "INSTALL FLASH PLAYER 6 AX.EXE" /Q


Silent install for Flash Player 6 for Netscape:


run "INSTALL FLASH PLAYER 6.EXE" /S


Silent install for Shockwave for IE and Netscape:


              run SHOCKWAVE_INSTALLER_FULL.EXE /S
