Note

When modifying your Abcpy.ini file, leave the 'SERIALNUMBER=' field blank and have the command line pass the SN through.

Example:

[OEM Install]

SERIALNUMBER=
INSTALLDIR=
DISABLEEMSFEATURES=1
DISABLEOLSFEATURES=1
STARTMENULOCATION=
SMARTSOUND=1

Command Line

Create file: application.xml.override
It must be in UTF-8 text format. 

It should look like this:

<?xml version="1.0" encoding="utf-8"?>
<Configuration>
            <Payload>
                        <Data key="EULA">Suppress</Data>
                        <Data key="Registration">Suppress</Data>
                        <Data key="Updates">Suppress</Data>
            </Payload>
</Configuration>

Edit file: Abcpy.ini

It should Contain:

[OEM Install]

DISABLEEMSFEATURES=1
DISABLEOLSFEATURES=1

Start installation with:
OEM.exe /UL1033 /V"SERIALNUMBER=xxxx-xxxx-xxxx-xxxx-xxxx-xxxx COUNTRY=177 DELETEDESKTOPSHORTCUT=1"

I am not sure if you need Country, and I don't remeber where I found the code for Norway which is 177.
