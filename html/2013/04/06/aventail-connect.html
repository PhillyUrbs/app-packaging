Deployment Tip Summary

Link to full Article: http://www.itninja.com/blog/view/deploying-the-sonicwall-aventail-vpn-client-with-customized-ini-settings Download the Aventail Connect client from your SonicWall Appliance - ngsetup64_en.exe -expand= (Ideally this path should be the same as the location of the main EXE file.) Once expanded, you will have 2 new files. "ngvpn.msi" and "ngsetup.ini" For unattended installation and configuration with your custom settings you will need to modify the INI file using the instructions from the SonicWall administrators guide. In this case it is 10.5.5. (Page 268, PDF here: http://www.sonicwall.com/app/projects/file_downloader/document_lib.php?t=PG&id;=520&dl;=1 ) Once your INI file is modified you are ready to distribute the package. Keep the customized INI file in the same folder as the EXE. Run the install with the following command "ngsetup64_en.exe -silent -f=ngsetup.ini" As long as the INI file is in the same folder as the installer you do not need to specify the full path to the INI file.
