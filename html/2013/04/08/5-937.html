Deployment Tip Summary

Register to download the MSI file for Enterprise Foxit Reader at the below link.  Registration is free and the software is free for personal or professional use.

http://www.foxitsoftware.com/pdf/reader/enterprise_reg.php

(Alternatively, you can start the install of the regular software on a test machine, then browse to %temp% and look for the MSI file.)

Download the MSI and use this command to install:

msiexec /i "EnterpriseFoxitReader542.0901_enu.msi" /qb- ALLUSERS=1 DESKTOP_SHORTCUT=0

Breakdown of the parameters:

  • /i - Install; specifies MSI path to install
  • /qb- - Quiet, force no GUI or response
  • ALLUSERS=1 - set ALLUSERS property; installs for all users
  • DESKTOP_SHORTCUT=0 - sets DESKTOP_SHORTCUT property to 0 so the shortcut under All Users is not created.  This parameter is the only one of these specific to Foxit; the rest can be used for any MSI installation.
2
Note

Get the MSI instead of the EXE because of ASK.com toolbar installations in the EXE.


(latest version (02-10-2011) http://cdn01.foxitsoftware.com/pub/foxit/reader/desktop/win/5.x/5.0/enu/FoxitReader502.0718_enu.msi )


Then run it silently:

msiexec /i FoxitReader502.0718_enu.msi /qn


Contrary to the silent EXE installation this MSI WILL NOT install ASK.com toolbar and homepage in IE. (tested version 502.0718)
