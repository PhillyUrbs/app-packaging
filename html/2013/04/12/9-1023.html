Note

To use the MSI you must extract it from the eCopyDesktop.exe. The eCopyDesktop.exe is located on the CD under the following directory structure: <CD root>\<Language>\eCopy Desktop.

Steps to extract MSI


1. Execute the ecopyDesktop.exe file by double clicking it.


2. When the eCopy Desktop Welcome dialog appears the MSI file is extracted to c:\windows\downloaded installations. Sort on the date modified and the folder with the latest time stamp is the one with the extracted MSI.

Note: The folder name will be a GUID.


3. Copy the MSI file to your distribution share.


Refer to the Command Line section for a list of properties.


If you need to install this on a Citrix environment that is redirecting the system drive to some other letter other than C:, you need to perform the following or you will get an error code of "1327 Invalid Drive C:\".

- Modify the DESKTOPSTORAGE property via the command line or create a transform. Example DESKTOPSTORAGE="%ALLUSERSPROFILE%\Documents"


Note: DO NOT modify the MSI directly as this is not recommended and supported by eCopy.


Below is some additional information from the eCopy's install manual.


"For some 'per-machine' deployments, a reboot may be necessary to enable full functionality with the eCopy Desktop Printer. Therefore, we recommend rebooting all systems that receive this deployment.


If you would like to configure your eCopy Desktop installation any further, launch eCopy Desktop from any machine. Configure it by clicking Options > Preferences in the main menu. All your preference changes are reflected in the registry key settings.

When youâ??ve made all your changes, capture this registry key as a .reg file:

HKEY_CURRENT_USER\Software\eCopy\Desktop9


Once you create this .reg file, you can distribute it to end points accordingly using Windows Scripting Group Policy, for example."

Command Line

I have not had much luck extracting the MSI from the CD.  But what I did find that was very useful was that if you browsed the CD to the following path: "Desktop\EN\eCopy Desktop" and ran the "eCopy Desktop 9.2 English.exe" with a /? switch, it provided the following command line parameters to use.


/L language ID

/S Hide intialization dialog.

For completely silent installs use:  /S /v/qn

          (No space between /v/qb)

/V parameters to MsiExec.exe

/UA<url to InstMsiA.exe>

/UW<url to InstMsiW.exe>

/UM<url to msi package>
