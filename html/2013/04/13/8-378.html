Command Line

There are several command line switches you can use with Netsetup.exe and Setup.exe.


/Loglevel=[level] = Specifies the level of logging to be done by the Setup program. Specify one of the following levels: 0 = Fatal errors, 1 = Critical errors, 2 = Warnings (default setting), 3 = Debug messages, 4 = Informational messages , 5 = Success messages


Each level includes the messages or errors of the lower levels. The default logging level is 2, which logs warnings, critical errors, and fatal errors. Please note, log level can also be set at time of distribution with Corel Distribute Manager.
