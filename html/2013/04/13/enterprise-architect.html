Note

When deploying the software silently, you might want to add the following user settings as well (i.e. via Active Setup)...

 

If using a Shared Key, the information for this is stored in the registry (export the settings after doing a manual installation):

[HKEY_CURRENT_USER\Software\Sparx Systems\EA400\EA\OPTIONS]
"SKT"=dword:00000001
"SSKSAddress"="ssks://your_server"
"SSKSPassword"="xxxxxxx"

Additionally, the user needs the key.dat from %appdata%\Sparx Systems\EA

 

To prevent the customization wizard from running when first starting the application, set this Registry value:

[HKEY_CURRENT_USER\Software\Sparx Systems\EA400\EA\OPTIONS]
"TRIAL_RUN_CUSTWIZ"=dword:00000000
