Command Line

Here is a command to install the MSI and it will allow you to specify a location for the MSI log file:
CiscoUnifiedPersonalCommunicatorK9.msi /qb! /lvemo C:\CUPCSetup\CUPCSetup.log

Notes:
- My setup script creates the log file folder before running the setup.
- The MSI does not install the version of Visual C++ or .Net Framework that are required for the setup.  You will need to run the executable to include these setup.
- To use the MSI my setup script checks for and install Visual C++ and .Net only if they are needed.

Script

This software adds plugin to Microsoft Outlook and other Microsoft programs.
You can use the commands below to kill the tasks for outlook and communicator if you have them in your environment.
taskkill /f /im outlook.exe
taskkill /f /im communicator.exe
