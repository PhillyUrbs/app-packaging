Deployment Tip Summary

8
Command Line

To extract the files to %temp%\visioviewer avoiding the EULA prompt:


visioviewer.exe /extract:%temp%\visioviewer /passive


Install silently:


visioviewer.exe /quiet


To then install silently and log (verbose) via the extracted .msi:


msiexec /i vviewer.msi /qn /LiV %temp%\vviewer2010.log


Install Visio Viewer 2010 along with any post-release updates that come out:


msiexec /i vviewer.msi /update KBxxxxx1.msp;KBxxxxx2.msp /qn /LiV %temp%\vviewer2010.log


Silently uninstall:


MsiExec.exe /x{95140000-0052-0409-0000-0000000FF1CE} /qn

Command Line

The following command line switches can be used on visioviewer.exe:


/extract:[path] - to extract the msi to the specified path

/quiet - silent mode

/lang:[locale ID] - specifies language

/norestart - prevents reboot whether prompted or not

/forcerestart - forces reboot on completion

/passive - no user interaction

/log:[path] - specifies log file path

Note

To install Service Pack 1, download it from here:
http://support.microsoft.com/kb/2460065


Run visioviewer2010sp1-kb2460065-x86-fullfile-en-us.exe /extract, and choose a folder.


Grab the .msp from the folder and put it in the same directory as vviewer.msi and vviewer.cab.


Install using msiexec /i vviewer.msi /update vviewersp1-en-us.msp /qn
