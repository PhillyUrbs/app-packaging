With Novell Client 2.0 SP1 (IR9a) I solved the silent Installation on Windows 7 with two things.


1. Install the Novell Network Client with devcon before you execute the setup.exe:

"devcon.exe dp_add netnvcli.inf"


2. Edit the install.ini for your need and execute the setup.exe with command line parameter:

"setup.exe /ACU /NCPF:NovelClientW7.txt /L:DEUTSCH"


For more Details read the Novell PDF "vista_client_admin.pdf".

You get devcon.exe for Windows 7 from the Microsoft Driver Develop Kit (WinDDK).
