Command Line

"setup.exe" command switches:

  • /D Default installation path

  • /E Serial number (e.g. "1234ps")

  • /C Company name

  • /V Server name (IP address or hostname; For network licenses only)

  • /S Silent mode

Example:

setup.exe /D "c:\local\installation\path" /C "My company" /E 1234PF /V servername /S

Command Line

It is possible to install PIPE-FLO to all client workstations from the server by using the command line tool. 

To utilize a command line install, open a command window and browse to your PIPE- FLO disc or Downloaded Media Files.
 

To install in Silent Mode, execute the following command: 

YourInstallerLocation\PIPE-FLOProfessional12Setup.exe/S

 

To install into a Specific Directory, execute the following command: 

YourInstallerLocation\PIPE-FLOProfessional12Setup.exe/D=DestinationPath


To install in Silent Mode AND into a Specific Directory, execute the following command: 

YourInstallerLocation\PIPE-FLOProfessional12Setup.exe/S/D=DestinationPath


PLEASE NOTE: The parameter order matters.

If you choose to do both modes, the/S parameter must be first and the/D parameter must be second.



The previous commands are case and space sensitive! 

Make sure that you capitalize your parameters properly and do not put quotes around your command even if it contains spaces.

Note

Commuter Licensing using Sentinel SDK

 


Commuting a License


This is for Version 12 only. This is also found in the, PIPE-FLO Install Instructions.pdf, document.


The first 3 Steps are required so the Server recognizes the remote client's Fingerprint. After the Server obtains the remote client's Fingerprint, it is not required to do the first three steps for that remote workstation in the future since the Server already has it's fingerprint. However, all 5 steps will need to be repeated for all new remote workstations that have never received a commuted license before.


When the commuted license exceeds the expiration date, the license is automatically returned to the server. 


Commuted backups are copied to the following directories:


  • 32-bit: C:\Program Files\Common Files\Aladdin Shared\HASP\

  • 64-bit: C:\Program Files (x86)\Common Files\Aladdin Shared\HASP\



 STEP 1: Obtain Fingerprint ID from Remote Computer


  • Login to Remote Client

  • Find the Computer’s Name, you will need this for later. (See: Control Panel > System and Security > System)

  • Install 'PIPE-FLO Professional 12 Setup.exe' to Remote Client (requires admin)

  • Once installed, open the ’License Utility’ tool: Start Menu > Engineered Software > Pipe-Flo Professional 12

  • Change to the ’Transfer License’ tab.

  • In the ’Collect information about the recipient computer’ box, click the [...] button.

  • Save the .id file.

  • Click ’Collect and Save Information’ button

  • Copy and save that .id file to the License Server

STEP 2: Enable Commuting


  • Login to the License Server

  • Open a web browser and go to: http://localhost:1947/_int_/config_detach.html

  • Ensure ’Enable Detaching of Licenses’ is checked

  • Click Submit

STEP 3: Apply Client ID File to Server


  • While still logged in to the License Server

  • Open a web browser and go to: http://localhost:1947/_int_/checkin.html

  • Click Button: Browse

  • Find and open the .id file that you saved in STEP 1

  • Click Button: Apply File

STEP 4: Create Commuted License File


  • Login to the License Server

  • Open a web browser and go to: http://localhost:1947/_int_/products.html

  • Find your PIPEFLO_NETWORK_V12_001 in the list, and Click button ’Detach/Extend’

  • Select Detach Method: Offline

  • Select Remote Client (this should be the Computer Name acknowledged in STEP 1)

  • Specify expiration date, and add any comments if necessary

  • Click Button: Detach & Attach

  • Click Button: Save as...

  • Save and copy the .h2r to the Remote Client.

STEP 5: Apply License to Remote Computer


  • Login to the Remote Computer

  • Open the ’License Utility’ tool: Start Menu > Engineered Software > Pipe-Flo Professional 12

  • Change Tab To: Apply License File

  • Next to Update File, click button: [...]

  • Select the .h2r file that you saved in Step 4

  • Click Button: Apply Update

Your license should now be successfully commuted. For more help, go to http://localhost:1947 in your web browser and select Help under Options.


Returning a Commuted License Before It's Expiration Date


Below are instruction to re-apply an attached license. If you have not already applied the commuted license to the remote computer, please do this first (see above).


  1. Log into the Remote Client

  2. Open a web browser and go to: http://localhost:1947/_int_/products.html

  3. Find PIPE-FLO_NETWORK_V12_001 and click Action: Cancel Feature

  4. Click the "Save As..." button

  5. Save and copy this .r2h file to the server

  6. On the server, go to: http://localhost:1947/_int_/checkin.html

  7. Click the "Browse" button and select the .r2h file you have saved

  8. Click button: Apply File

You should now have your license available for use, or re-commuting. To double-check, go to Products and see the Available column to see how many available licenses that you have.
