Deployment Tip Summary

Using the ability to create a answer file that is included with the software you can make the changes needed to give this a quick and efficient install. If you need to install additional software with this it may be best to use a bat file rather than trying to integrate it into the install.

1
Note

For Office we end up creating an admin file with all the settings we need. You can create one by running the setup.exe with the /admin switch. Then we save the admin file with the installation files, zip it up and upload it into KACE.

Our installation command is set to configure manually and looks something like this "setup.exe /adminfile adminfilename.MSP" and we choose "Don't Prepend msiexec.exe" and to "Delete Downloaded Files". So far this has worked well and successfully deployed to over 280 machines.
