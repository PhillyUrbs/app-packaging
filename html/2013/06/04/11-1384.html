Note

This is the folder structure for COREL PAINTSHOP PRO XI


0x0409.ini

Data1.cab

PSP1120_EN_DE_FR_ES_IT_NL_CORELRETAIL.msp

psppxi.msi

1033.mst

Setup.ini


(To make the setting as 'No' in Help->Message Setting ->No)


Here the problem is the vendor given MST is working fine until we include  following file

C:\Documents and Settings\All Users\Application Data\Corel\Messages\540228105_807001\Message.policy

in Alluser Profile. So create new MST using install Tailor in Wise Package Studio. Copy the above Message.policy file from the currently loged in user and paste it in that location and


add this file in to your mst. it will create on Cab file.


Then the folder structure will be like this


0x0409.ini

Data1.cab

PSP1120_EN_DE_FR_ES_IT_NL_CORELRETAIL.msp

psppxi.msi

1033.mst

Setup.ini

psppxi_WISETRFM_2.cab



To suppress EULA window


Open your MST using Wise and goto regsitry section and select All Features and select the following hive HKLM\Software\Corel\Paint Shop Pro\11

Here in the right side the following two registry entries will be there

ShowEula=0 (DWORD)

ShowEula=1 (DWORD)


Delete those two entries and select "PaintShopPro" Feature and create one registry entry in that same location ShowEula=0 (DWORD)



These are the properties we have to add in the MST


SERIALNUMBER= XXXXXXX-XXXXXXX-XXX

C_LAUNCHAPP =0 (To suppress app launch)

C_INAPPYAHOO=0

C_YAHOOTOOLBAR=0 (To avoid yahoo based installs)

C_INSTALLSNAPFIRE=0 (To avoid initializing snapfire feature install)

ISCHECKFORPRODUCTUPDATES=0 (To disable update)

C_INSTALLRCM=0

C_READMESHORTCUT=0 (To remove readme Shorcut from start menu->programs->corel paintshop pro XI folder)

C_SHOWCREATEQUICKLAUNCH=0 (To uncheck the checkbox which is creating QuickLaunch Shorcut)

C_SHOWINSTALLSNAPFIRE=0

C_PDFSHORTCUT=0

C_UNINSTALLSHORTCUT=0 (To remove uninstall Shorcut from start menu->programs->corel paintshop pro XI folder)

C_CREATEQUICKLAUNCH=0 (To remove QuickLaunch Shorcut)

C_APPLAUNCHREGISTRATION=0 (To stop registration window upon launch)

C_UPDATESERVICE=0 (To disable update services)

RebootYesNo No


If you want to delete the desktop Icon, add the following property in MST

C_CREATEDESKTOPICON=0



To remove the Learning Centre window (After Launching the application you will get one window on left side) from the package we have to add following registry in MST


[HKEY_CURRENT_USER\Software\Corel\Paint Shop Pro\11\UI Customization\BCGPBaseControlBar-59641]

"IsVisible"=0 (DWORD)


To apply the current user registry add StubPath in MST


HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Active Setup\Installed Components\[ProductCode]


StubPath=msiexec /f [ProductCode] /qn (String)

Version=1 (String)
