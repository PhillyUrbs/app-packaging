Note

You may pre-configure all user settings and distribute them to all clients. This is very easy using the".12g" settings file. In the options tab you may save the current settings to create .12g settings file. It will import settings automatically if it finds a file named "settings.12g" in the same path as the installation file.


To import settings later just call the .12g file from a script or shortcut. Use "/s" for stealth operation. For example:


\\myserver\thefolder\settings.12g /s


Source: 12Ghosts FAQ

Command Line

/silent  = For unattended, silent installation.


/nostart = Don't start the program after setup is complete. By default the program is started. For the packages, 12tower.exe is started by default. If there were Ghosts running before setup those are restarted.


/noshort = No shortcuts are created in the Start-Programs menu. By default shortcuts are created.


/language:N = The initial language used for setup. The N-th entry in the drop-down list.


/path: = Installation path. Default is the folder where the program was installed last time. If this is the first installation the subfolder "12Ghosts" is created in the system's "Program Files" folder. This usually is at the root of the same partition as the Windows folder.
