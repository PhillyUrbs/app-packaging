A9CAD is a FREE 2-Dimensional ACAD application.  A9CAD will read/write .dwg files.


To obtain a .msi for the application use a /A switch on the downloaded .exe installer.


The A9CAD software seems to have a problem with opening files through file associations.  A9CAD program will launch but will not open the .dwg file that was launched.

The setup file from A9Tech.com is an InstallShield Installscript .exe.  Use a /? to obtain a list of command lines.


---------------------------

InstallShield

---------------------------

Command line parameters:


/V parameters to MsiExec.exe

/S Hide intialization dialog.  For silent mode use: /S /v/qn.

/P password mode

/A administrative installation                       

/J advertise mode

/X uninstall mode

/UA<url to InstMsiA.exe>

/UW<url to InstMsiW.exe>

/B cache installation locally


Using the /A will give you a native Windows Installer .msi.
