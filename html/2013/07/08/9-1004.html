Note

If you have been supplied the correct media you can create a MST following the notes in WinDVD9_DeploymentGuide.pdf


If you have downloaded the media you will need to record an iss file and create a transform.


Call the response file setup.iss and copy it to same location as setup.exe


To install use following command line:

setup.exe /s /L1033 /V"TRANSFORMS="CRL-WinDVD-9-R01-3018.mst""


The MST contained following properties\registry:

SERIALNUMBER=XXXXXXX-XXXXXX-XXX

C_INSTALLRCM=0

C_APPLAUNCHREGISTRATION=0

C_SHOWUPDATES=0

C_CREATEDESKTOPICON=0


HKCU\Software\InterVideo\DVD9 "ShowEULA"=dword:00000000


I could not find any properties to stop the installation of GoogleToolBar, QuickTime or RealPlayer so I just deleted the 3rdpartyapp folder which contained the installers.

Note

Tested with WinDVD 9 Corporate Edition:

1. Create a administrative installation --> setup.exe /a

2. Remove the folder "3rdpartyapp" from the source, if you wan't install 3rdparty tools

3. Run Setup.exe /r to create the setup.iss answer file --> Copy the file "C:\Windows\setup.iss" to the WinDVD administrative installation point (overwrite existing)

4. Install silent with following command line command: "AddPath\Setup.exe" /s /sms /f1"AddPath\setup.iss" /v"/l*v "AddPath\WinDVD9Install.log""

5. For Uninstall use "C:\Program Files\InstallShield Installation Information\{E3993D46-AE3F-402E-9F9D-EEBDFBEC3564}\setup.exe" /s /sms /uninst


If you have a Msiexec.exe APPCRASH with Vista or Win7, remove following Custom Actions in the WinDVD.msi


- StopService.E0ED139E_909D_4EB4_9AFC_8C09B1718DDC

- UnRegService.E0ED139E_909D_4EB4_9AFC_8C09B1718DDC

- RegService.E0ED139E_909D_4EB4_9AFC_8C09B1718DDC
