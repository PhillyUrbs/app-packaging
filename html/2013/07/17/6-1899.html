Note

For unattended GroupWise installation the "setup.cfg" is the file used for unattended rollouts of the GroupWise client. The syntax is pretty much the same for both GW5.x & 6.x clients, however I separated them anyways. I still recommend reading through all of the following. The addon.cfg is what you can use to install C3POs (client enhancements) right along with the GroupWise client. WMS (windows messaging system) is included with gw6x for all OSes, but with 5.5ep & lower you have to preinstall ntwms.
