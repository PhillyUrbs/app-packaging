Deployment Tip Summary

Deploying the KACE agent isnt as difficult as many believe. Pushing, using a login script or manually installing are all methods of deployment. Several requirements are similar for all system management tools and should be considered standard. An account with admin rights to install software, file sharing enabled (SSh on Mac and linux) and UAC (or simple file sharing on XP) disabled for the deployment. Firewall rules open for ports 80,443 and 52230 for the agent.

-1
