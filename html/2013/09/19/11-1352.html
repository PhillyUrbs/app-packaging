Note

When you install AER, the GUID will be dependent on the application using it. 


For example, if you want to install AER as a pre-requisite for Application Virtualization 4.5, then you need to use:


dw20shared.msi APPGUID={FE495DBC-6D42-4698-B61F-86E655E0796D} ALLUSERS=2 REBOOT=ReallySuppress /qn


For Visual Studio 2010:


dw20shared.msi APPGUID={34C8AA92-6E3B-3890-8312-6156FE95F9BF} ALLUSERS=2 REBOOT=ReallySuppress /qn
