Note

Desc: software to schedule and manage Swimming meets.


Notes and CAVEATS:

-Must extract MSI from EXE by obtaining the MSI from %TEMP%

-a default setup.exe installation will only work for the user account that installed it. (work around below)

-Installs at the root of C: in C:\Hy-Sport folder by default. You can change it using INSTALLDIR property but the software is hard-coded to use a database in C:\swmeets folder.

-Cannot feed ALLUSERS=1 to Setup.exe, Must extract the MSI and run it directly:

"MEET MANAGER 2.0 for Swimming.msi" ALLUSERS=1 REBOOT=RS /qn /L*v %TEMP%\MEET_MANAGER_2_0.LOG


-Found an update to 2.0h.

-Installing it does not kill the ALLUSERS=1 effect


Windows 2000: Need to install jet35sp3.exe as prerequisite.


To run as NON-Admin, change file permissions on the folders:

SetACL.exe -on "C:\swmeets" -ot file -actn ace -ace "n:users;p:change"

SetACL.exe -on "C:\Hy-Sport" -ot file -actn ace -ace "n:users;p:change"

DL from http://setacl.sourceforge.net/


DRM issues:

License seems it must be installed from a CD-ROM. Copying the SWMEET.MEM file to the root of D: will not work.

Was able to install the license once w/o CD but could not replicate. MFR web site does not offer help/support.

There is no obvious way to hint to it where the .MEM file is located.
