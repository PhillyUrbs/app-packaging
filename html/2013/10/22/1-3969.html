Note

This works for the german version of Thunderbird 1.5.10, available at http://www.thunderbird-mail.de but should also work for the original version:


Install with

Thunderbird Setup 1.5.0.10.exe -ms


Uninstall with

%ProgramFiles%\Mozilla Thunderbird\uninstall\uninstall.exe -ms


If you want to create an MSI-Packet, you should build an embedded installer. To do this, you insert the installer as a binary into the packet. Then create two Custom Actions for executing the binary installer and one for executing the uninstaller.

Note

:::Integration of Lightning (Calendar Add-On) into Thunderbird:::


*Copy Add-On in program folder*


copy /Y "lightning-0.9-tb-win.xpi" %ProgramFiles%\Mozill~1

*change into folder, then use following syntax:*


thunderbird.exe -install-global-extension "lightning-0.9-tb-win.xpi"
