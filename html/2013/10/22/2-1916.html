Command Line

For Thunderbird 2.0.0.0 [de] and 2.0.0.4 [de]:

- Download Setup

- Extract Setup into a folder (e.g. with WinRAR)


INSTALLATION

============

Start "setup.exe /S"

=> Setup will start installing Thunderbird with default preferences.


PREFERENCES

===========

Want to change something? nonlocalized\defaults\pref is the folder of choice. Edit the .js files contained within this folder to change default preferences


UNINSTALL

=========

execute

[program files folder]\Mozilla Thunderbird\uninstall\helper.exe /S

SET AS DEFAULT MAILER

=====================

Set Thunderbird as the default mail client:

"C:\Program Files\Mozilla Thunderbird\uninstall\helper.exe" /SetAsDefaultAppGlobal

Note

Autoconfiguration is helpful for not having to enter usernames and servers, and can be done using Mozilla's MCD (Mission Control Desktop).


Information is available here:
http://mit.edu/~thunderbird/www/maintainers/autoconfig.html


Good examples here:
http://www.int-evry.fr/s2ia/user/procacci/netscape/en/mozilla-autoconfig-en.html#htoc12

NB If you don't want to lock preferences you can substitute "defaultpref" for every instance of "lockpref".


And this looks like they've used the above as a reference:
http://developer.mozilla.org/en/docs/MCD,_Mission_Control_Desktop_AKA_AutoConfig


Also in there is LDAP related configuration, for example for getting a user's full name from just their current login account name. LDAP addressbooks can be configured from there also.
