Command Line

Create setup.iss

AStudio61sp1_1.exe /R /F1"C:\setup.iss"

Installation will be started, and setup.iss will be created in the given Directory. In the setup.iss, all settings which are made in the Installation are saved.

To Run the setup on silent, type

AStudio61sp1_1.exe /S /F1"C:\setup.iss"

The settings from the setup.iss will be used

If installed silently, the USB Driver is automatically installed, but VS Isolated Shell has to be installed seperately before Atmel Studio

Command Line

Uninstallation: Use the setup.exe located in  "%ProgramFiles%\InstallShield Installation Information\{71D5A296-D77E-4BC5-BB75-12AD20001A61}"

setup.exe /s /f1"uninstall.iss" -remove_only -runfromtemp

I used an .iss-File that I created via

setup.exe /R /F1"C:\uninstall.iss"

Repair works just like Uninstall. Which one is done depends on what is written in the .iss-File
