Note

If you haven’t ORCA* in your computer, Download it here: http://www.clubic.com/telecharger-fiche71618-orca.html and install it.

*ORCA is editor of Windows Installer Files (MSI)

 

Now, you must select the MSI of the application, here “Nuance PDF Converter 7” in folder “Setup”.

Right click on a MSI and select “Edit with ORCA”.

 

 

 

 

License Agreement

For accept the licence Agreement, you must modify this value of the property in the Tables Property:

o    AgreeToLicense     =          Yes

 

Customer Information

For adds the “User Name”, “Organization” and “Serial Number”, you must create this Property in the Tables Property:

o    COMPANYNAME   =          MSK

o    USERNAME           =          TOOLS

o    ISX_SERIALNUM   =          XXXXX-XXX-XXXX-XXXX-XX

 

For adds Property select in “Tools Bar”, “Tables” and “Add Row…”

 

Setup Type and Custom Setup

The “Custom Setup” option are:  




























“Custom Setup” Option

MSI Property

Open PDF/XPS Files in Microsoft Word

SSWORDADDIN

Open PDF/XPS files in Microsoft Excel

SSEXCELADDIN

Open PDF/XPS files in Microsoft PowerPoint

SSPPTADDIN

Convert PDF/XPS files attached to messages in Microsoft &Outlook

SSOUTLOOKADDIN

Enable Nuance PDF toolbar in Microsoft Office

SHOWOFFICEADDINICON

&Download and unlock PDF/XPS files from Internet Explorer links

SSIEINT

Direct conversion from PDF/XPS files

SSSHELLEXT

Lotus Notes Integration

SSLOTUSADDIN

WordPerfect Office support

SSWPADDIN

FireFox Integration

ISINSTALLFIREFOXADDIN

Desktop Shortcut

SSDESKTOPSHORTCUT

Quick Launch toolbar Shortcut

SSQUICKLAUNCHSHORTCUT

 

For customize components of PDF Converter, you must modify the “Setup Type”, for that go to Tables “Property” and modify this “Value” of this property:

o   SetupType                       =          Custom

o   _IsSetupTypeMin            =          Custom

 

For disable Components in “Custom Setup”, you must Change “value” in Property “Checkbox”:

o   ISINSTALLFIREFOXADDIN = 0

 

And delete Property in Tables “Property”

 

Additional option

For disable Registration Product modify “PROD_ACT” the property in Tables “Property”

o   PROD_ACT                     =          0

 

It willmakethe product registrationat first launchof the application

 

For disable update product, you must delete all value of “InstallShield Update Service Setup Player” far that go to the Tables Property “AppSearch” and select text after “sigShdocvw.” here D5A09942_925E_44C9_979D_D78D19ABF629 and search it in all tables Property et delete.

 

Create you transforms MST

Select in the “Tools Bar”, “Transform” and select “Generate Transform…”
