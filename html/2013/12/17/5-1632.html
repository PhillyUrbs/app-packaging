Note

GaussView doesn't have a working silent installer. However, you can install it manually then create a zip file with the contents of C:\G09W. The install directory is the same as Gaussian 09W proper. By installing Gaussian 09W first, then extracting the GaussView files over top, you can achieve a silent install.


To register Gaussview on x64, create a reg file with the following contents:


Windows Registry Editor Version 5.00


[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Gaussian, Inc.\GaussView\5.0]

@=""

"Serial No."="XXX############X-####X"

"Path"="c:\\g09w"

"Version"="5.0"


[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths\g09w.exe]

"Path"="C:\\G09W"

@="C:\\G09W\\g09w.exe"
