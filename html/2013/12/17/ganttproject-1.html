Command Line

This installs with just a simple command line:


ganttproject-2.#.#.exe /S (uppercase S is important)


I do not see a native msi version on the site http://ganttproject.biz/

note - this is a java app, so make sure you have the latest sun jvm installed (unattended of course!)


What is Ganttproject? GanttProject is a free and easy to use Gantt chart based project scheduling and management tool. Our major features include:

        * Task hierarchy and dependencies

        * Gantt chart

        * Resource load chart

        * Generation of PERT chart

        * PDF and HTML reports

        * MicroSoft Project import/export

        * WebDAV based groupwork
