Note

This is for Google Sketchup v3.0.3117

Turn off updates:
[HKEY_CURRENT_USER\Software\Google\SketchUp8\Preferences]
"CheckForUpdates"=dword:00000000

Turn Off Welcome dialogue:
[HKEY_CURRENT_USER\Software\Google\SketchUp8\WelcomeDialog]
"ShowOnStartup"=dword:00000000

Turn off the Instructor window:
[HKEY_CURRENT_USER\Software\Google\SketchUp8\SnappyInstructor]
"show"=dword:00000000
"Minimized"=dword:00000000

Advertised the shortcut (not advertised by default)
Create a new component with the keys above (set one as a keypath)
Attached to Feature SketchUpModule (which contains the component compSketchUpExe.78D17A5F_0E0A_44D2_877D_2C56D45D16B7)

Sort out the icons too, no icon in add/remove and when you advertised the shortcut it loses the icon.
