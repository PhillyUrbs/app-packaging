Note

First Step is to integrate the latest patch in the msi

msiexec /a Lync.msi TARGETDIR=C:\temp\AIP

Second Step is to integrate the latest patch in the msi created by the administrative install

   msiexec /p Lync.msp /a C:\temp\AIP\Lync.msi


Third and last Step is to create an MST with orca containing the custom secure property

   USEMSIFORLYNCINSTALLATION = #1

  

Got ya's :

Step 1 : You need to extract the msi from the setup.exe

Step 2 : Get the latest patch from microsoft download site

Step 3 : Use the #1 value (the # is really necessary)

Msi-repair = change the value of OCSETUPDIR in the Directory table to Directory_Parent = ProgramFilesFolder and DefaultDir = orv4xmit|Microsoft Lync
