Note

Uninstall options for McAfee Security Scan installed by Adobe products.


McAfee Security Scan is additional option when installing Adobe flash.  Ultimately when installing Adobe flash you would want to deselect the option to install the McAfee Security Scan.  This particular product is designed for consumer use.


Uninstall methods:


McAfee Security Scan uses the Nullsoft Installer NSIS 2.22 as installer. Therefore you can apply /S to the UninstallString defined in HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\McAfee Security Scan



Therefore "C:\Program Files\McAfee Security Scan\uninstall.exe" /S should do the trick.


Note that /S is case sensitive.


(Reference: https://community.mcafee.com/message/96700#96700)
