Note

1. See Package GhostScript 8.70 installation


2. run FreePDF4.02.EXE, this will extract installation files to:

C:\Documents and Settings\%username%\Local Settings\Temp\IXP000.TMP


3. start command line


4. c:\Documents and Settings\%username%\Local Settings\Temp\IXP000.TMP\fpsetup.exe /Q


Remember: There has to be an entry of GhostScript before installing FreePDF in HKEY_LOCAL_MACHINE\SOFTWARE\GPL Ghostscript\8.70

Note

If you want to create a MSI package of FreePDF and don't know how to add the printer then use Process Monitor and look for registry keys located under

HKLM\SYSTEM\CurrentControlSet\Control\Print

-Forms

-Printers

-Monitors

-Environments


also have a look at keys under

HKCU\Software\Microsoft\WindowsNT\CurrentVersion-PrinterPorts

-Devices


If you include all necessary keys regarding the FreePDF printer then your MSI should create the printer and you should be able to use all functions of FreePDF. Perhaps a restart of the system is needed but I haven't tested it yet.
