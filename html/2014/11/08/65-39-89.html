Note
  • open msi with Orca (http://msdn.microsoft.com/en-us/library/windows/desktop/aa370557(v=vs.85).aspx) and switch to "CustomAction" Table
  • edit "BuildInstallCommand"-Target

/silent /install "[ProductTag]" /installsource enterprisemsi[OptOmahaArgs] /appargs "appguid={8A69D345-D564-463c-AFF1-A69D9E530F96}&installerdata=%7B%22distribution%22%3A%7B%22msi%22%3Atrue%2C%22system_level%22%3Atrue%2C%22verbose_logging%22%3Atrue%7D%7D"

  • copy the text after installerdata= to clipboard ( except " )
  • open http://meyerweb.com/eric/tools/dencoder/  and paste the text
  • click decode
  • add your own settings ( master_preferences )
  • click encode
  • copy the text to clipboard
  • paste the text after installerdata= and make sure there is " at the end
/silent /install "[ProductTag]" /installsource enterprisemsi[OptOmahaArgs] /appargs "appguid={8A69D345-D564-463c-AFF1-A69D9E530F96}&installerdata=%7B%0A%20%20%22homepage%22%20%3A%20%22http%3A%2F%2Fdev.chromium.org%22%2C%0A%20%20%22homepage_is_newtabpage%22%20%3A%20true%2C%0A%20%20%22distribution%22%3A%20%7B%0A%22msi%22%3Atrue%2C%0A%22system_level%22%3Atrue%2C%0A%22verbose_logging%22%3Atrue%2C%0A%20%20%20%20%20%22create_all_shortcuts%22%3A%20true%2C%0A%20%20%7D%0A"
 
  • save file as msi or generate transform.
