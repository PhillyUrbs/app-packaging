Note
Here's a quick way to install version 11 silently...

1. Obtain the .EXE file as from www.real.com/player
2. Navigate to the logged in user's Temp folder (for example: C:\Documents and Settings\JoeUser\Local Settings\Temp or C:\Users\JoeUser\AppData\Local\Temp) and clear out any existing folders in order to make things easier to locate in the next step.
3. Launch the downloaded RealPlayer11GOLD.exe installation, but do NOT install the program (just leave it running at the initial installation window).
4. Open the newly created rninst~x temp folder, wait until the RealPlayer11GOLD.exe file is fully available, and then copy the RealPlayer11GOLD.exe file to another location.
5. Cancel the RealPlayer installation setup.
6. Use the "/s" switch to install the acquired .EXE silently.

A lot of the other settings and preferences are similar to the previous version (be aware that they may have moved slightly or been renamed)...

Command Line
uninstalling RealPlayer unattended silent

If you use the uninstall string of the registry with a batchfile, you will get a dialogbox when it's ready. There is no parameter to supress the message. But you can do this:

1. Create this shortcut: "C:\Program Files\Common Files\Real\Update_OB\r1puninst.exe" RealNetworks|RealPlayer|6.0 -s

2. Execute the shortcut with your uninstall-batch-file

3. ping 127.0.0.1 -n 100 (Because your batch will go immediately to the next step, you must make sure, that at first the uninstall is ready)

---------------------------------------------------------
GERMAN TRANSLATION:

Deinstallation RealPlayer

Führt man den Uninstall-String aus der Registry über eine CMD aus, erhält man am Ende eine Message-Box, die bestätigt werden muss. Es gibt leider keine Option, den Dialog zu unterdrücken. Man kann dies allerdings folgendermaßen umgehen:

1. Folgende Verknüpfung erstellen: "C:\Programme\Gemeinsame Dateien\Real\Update_OB\r1puninst.exe" RealNetworks|RealPlayer|6.0 -s

2. Die Verknüpfung über die Deinstallations-Batch starten

3. ping 127.0.0.1 -n 100 (Da die Batch nach Aufruf der Verknüpfung sofort zum nächsten Befehl springt, muss sicher gestellt werden, dass erst die Deinstallation abgeschlossen wird.)

Note
Alright I finally found a process to run Real player 11 Gold silently with limited over-head.

First download Realplayer11gold.exe from real.com

run Realplayer11gold.exe

when the EULA prompts to accept, navigate to
C:\Documents and Settings\%username%\Local Settings\Temp

from there, copy over any rinst* directories to a network or local install point.

one of the rinst directories will have an application called RealPlayer11Gold.exe(Check the properties of the .exe object and go to the general tab. the description should say "Shell executable of Setup program". if it doesnt say that, it is the wrong executable).

run the correct RealPlayer11Gold.exe from the command line with the /s switch and it will install silently.

Hope this Helps!!
