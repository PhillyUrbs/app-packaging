Note
To disable the automatic startup of Skype with the user logon, remove the line from the Registry table, which contains the following entry in the key column:
Software\Microsoft\Windows\CurrentVersion\Run

To prevent Skype from becoming a Supernode, add this line to the Registry table:

HKLM1 2 SOFTWARE\Policies\Skype\Phone DisableSupernode #1 SkypeRegistryAllUsers

To disable search for automatic updates, add this line to the Registry table:

HKLM2 2 SOFTWARE\Policies\Skype\Phone DisableVersionCheck #1 SkypeRegistryAllUsers
