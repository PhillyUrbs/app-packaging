Note
Silent install-

First you will need to extract the .CAB files from the downloaded executable.

Perform the following:

1. Start Run %temp% or however you want to get to the Temp folder

2. Run the downloaded executable (VMware-workstation-full-7.0.0-203739.exe)

3. Once you run the executable you will see a folder named something like (vmware_1257284968) in the temp folder. This is where Workstation will extract all of the .CAB files to along with vmware workstation.msi.

4. Copy the vmware_1257284968 folder to another location AFTER the installer is done extracting all of the .CAB files, but BEFORE you click Next on the installer. (If you complete the install before you copy the files off they will be deleted).

5. Click Cancel on the installer after you have copied off the .CAB files.

6. Run the vmware workstation.msi file with the appropriate msiexec switches. In my case I ran it quiet with the basic UI or /qb

Note
To remove EULA/updates, you can make a "preferences.ini" file, and put it in a central location, and distribute after installation is complete.
(Best way is to install it on one machine first, to make shure you get the right variables for versions.)

In the Preferances.ini file include the following lines to disable eula and updates for Workstation, and Player:
NB: Remember to change BuildNumber according to your version!

______
pref.eula.0.appName = "VMware Workstation"
pref.eula.0.buildNumber = "227600"
pref.eula.1.appName = "VMware Player"
pref.eula.1.buildNumber = "227600"
webUpdate.updatesAvailable = "FALSE"
webUpdate.enabled = "FALSE"
pref.downloadPermission = "deny"
pref.vmplayer.webUpdateOnStartup = "FALSE"
______


Then use a script like the one following to copy the Prefs file in after installation (ran in user context ofcourse):

______
SET SOURCE=\\folder_where_you_have_the_preferences.ini
if not exist "%APPDATA%\VMware" mkdir "%APPDATA%\VMware"
if not exist "%APPDATA%\VMware\preferences.ini" copy "%SOURCE%\preferences.ini" "%APPDATA%\VMware\"
______
