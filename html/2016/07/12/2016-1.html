Command Line
To install Access Runtime 2016 with no UI or reboot (similar to 2010 version):

1. Extract files usingaccessruntime_4288-1001_x86_en-us.exe /extract:c:\temp\access2013rt

2. Create config.xml:

<Configuration Product="AccessRT"> 
<Display Level="None" CompletionNotice="no" SuppressModal="yes" AcceptEula="yes" /> 
<Logging Type="standard" Path="C:\Windows\Temp\" Template="Microsoft_Access_2016_Runtime_Setup(*).log" /> 
<COMPANYNAME Value="Company Name" /> 
<Setting Id="SETUP_REBOOT" Value="Never" /> 
</Configuration>

3. Run setup.exe /config config.xml
