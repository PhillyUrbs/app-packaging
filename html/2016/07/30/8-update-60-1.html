    Tip: Default values do not need to be called. WEB_JAVA_SECURITY_LEVEL=H is a default value. SPONSORS=0/1 is not supported in the offline installer according to the Oracle documentation as well. OEMUPDATE is deprecated and no longer applicable. The rest looks good.
