Note
First create a InstallShield-Script, the "setup.iss" file by launching the setup.exe with the switch "/r". you can now click through the setup and set your prefered settings. after hitting the finish button you´ll find the "setup.iss" in your Windows directory.

For a complete silent installation of PowerDVD 6 you have to start the setup with the created answer-file, after that set some MACHINE-based reg-infos, then some USER-based reg-infos.

1) SETUP

Command: "X:\PowerDVD_6\Setup.exe /S /SMS /f1X:\PowerDVD_6\SETUP.ISS /f2C:\PowerDVD6.log /v/qb-"

/S for "commands for MsiExec follows"
/SMS for "Setup.iss follows"
/f1 for the path to the setup.iss
/f2 for the path to the logfile which will be created by Setup (should be local, if a network source is given here and the user doesn´t have rights to write to, setup will fail!)
/v for commands for MsiExec
/qb- for Quiet / no User-Interaction / only progressbar provided

2) MACHINE-REGKEYS

Command: "RegEdit.exe /s X:\PowerDVD_Machine.reg"

Contents of PowerDVD_Machine.reg:

[HKEY_LOCAL_MACHINE\SOFTWARE\CyberLink\PowerDVD]
"UI_PMK"="*** YOUR-SERIAL-HERE ***"
"GetBy"="CDKEY"
"UI_NonTrial"=dword:00000001
"CDKey"="*** YOUR-SERIAL-HERE ***"
"KeyUpdate"=dword:00000001
"MustUpdateKey"=dword:00000000
"HasCDKEY"=dword:00000001

3) USER-REGKEYS

Command: "RegEdit.exe /s X:\PowerDVD_User.reg"

Contents of PowerDVD_User.reg:

[HKEY_CURRENT_USER\SOFTWARE\CyberLink\PowerDVD]
"UI_PMK"="*** YOUR-SERIAL-HERE ***"
"GetBy"="CDKEY"
"UI_NonTrial"=dword:00000001
"CDKey"="*** YOUR-SERIAL-HERE ***"
"KeyUpdate"=dword:00000001
"MustUpdateKey"=dword:00000000
"HasCDKEY"=dword:00000001
