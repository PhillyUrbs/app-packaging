No product Activation:
To disable the Prompt to Activate your product after the start of Acrobat 9, you must first Select "ask never again".
Now a file called "com.adobe.118.registration.reg" will be created under "%userprofile%\applicationdata\Adobe"
This file, in this Path, suppress the "Activation Window".

To Disable the AutoUpdate functions you can delete this files from your computer:

Adobe_Updater.exe
UpdaterInstallMgr.exe

They could be found by the search function
in different folders.

You can set Registry Keys to deactivate functions:

[HKEY_LOCAL_MACHINE\SOFTWARE\Adobe\Adobe Acrobat\9.0\AdobeViewer]
"Launched"=dword:00000001
"EULAAcceptedForBrowser"=dword:00000001
"EULA"=dword:00000001
"Registered"=dword:00000002

Note
Feature lockdown is not supported on the Macintosh platform.
While the application preference is stored in HKCU, its editability is controlled by it boolean lockdown counterpart in HKLM. The hierarchy within FeatureLockdown is the same as the one under the HKCU Security directory. All preference values in FeatureLockdown are booleans that indicate their editability. When marked as uneditable, the application user interface item associated with that preference is disabled.

To lock down features, navigate to:
7.x: HKEY_LOCAL_MACHINE\SOFTWARE\Adobe\<product>\<version>\FeatureLockDown
8.x: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\<product>\<version>\FeatureLockDown

Note
How to resolve "User has entered a retail serial number in a volume build. Rejecting." in addition to other efforts.

1) Create an AIP.
2) Patch it to newest Version, which is 9.3.2 at the Moment.
3) Copy all *.ini and .mst files from installation media to AIP folder.
3) Install Adobe Customization Wizard 9 and open Asnv.tmp.
4) Make your settings and SAVE the package, don't generate a transform file.

setup.ini shall now look like this:

[Startup]
RequireOS=Windows XP
RequireMSI=3.0
RequireIE=6.0.2600.0
CmdLine=/sl"1031" /spb /rs

[Product]
msi=AcroStan.msi
Languages=1033;1031;1036
1033=English (United States)
1031=German (Germany)
1036=French (France)
CmdLine=TRANSFORMS="Asnv.mst"

[Windows XP]
PlatformID=2
MajorVersion=5
MinorVersion=1
ServicePackMajor=2

[MSI Updater]
Path=WindowsInstaller-KB893803-v2-x86.exe


Launch setup via

msiexec.exe /i "%SRC%\AcroStan.msi" TRANSFORMS="%SRC%\Asnv.mst" REBOOT=ReallySuppress /l "%App%\install.log" /qb-!

This works fine for me except some little configurations to do.

Note
Some registry values documented in this KB entry did not work in my case. I installed Adobe Acrobat 9.4 with all incremential updates. The following reg values and settings worked:

Hide "Improvement Program Options..." in menu item "Help"
HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\Adobe Acrobat\9.0\FeatureLockDown]
"bUsageMeasurement"=dword:00000000

Hide "Digital Editions" in menu item "Help"
[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\Adobe Acrobat\9.0\FeatureLockDown]
"bShowEbookMenu"=dword:00000000

Hide "Updater" in menu Edit - Preferences and hide "Check for updates..." in menu "Help"
[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\Adobe Acrobat\9.0\FeatureLockDown]
"bUpdater"=dword:00000000

Disable automatic updates
[HKEY_LOCAL_MACHINE\SOFTWARE\Adobe\Adobe ARM\1.0\ARM]
"iCheck"=dword:00000000

Hide "Collaborate" in menu "File"
[HKEY_CURRENT_USER\Software\Adobe\Adobe Acrobat\9.0\Workflows]
"bEnableAcrobatHS"=dword:00000000
"bEnableWorkflowPart"=dword:00000000
"bEnableRTCPart"=dword:00000000

Disable Adobe Java Script in "Preferences"
[HKEY_CURRENT_USER\Software\Adobe\Adobe Acrobat\9.0\JSPrefs]
"bEnableJS"=dword:00000000

Disable "Allow multimedia operations" in Preferences - Multimedia Trust (legacy)
This settings are stored in the 3 files SharedDataEvents,TMDocs.sav,TMGrpPrm.sav in the folder %appdata%\Adobe\Acrobat\9.0

To do not display the EULA and registration dialog at startup set the two properties
EULA_ACCEPT=YES REGISTRATION_SUPPRESS=YES
with the installation command line.

With the commandline property REMOVE_PREVIOUS_READER=YES all previous Adobe Reader versions will be uninstalled. But in my case when I started Acrobat the first time the MSI maintenance dialog is shown and Acrobat is reinstalled again.
