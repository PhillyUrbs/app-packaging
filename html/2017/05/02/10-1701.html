Command Line

get the MSI from here   ftp://ftp.adobe.com/pub/adobe/reader/win/10.x/10.1.0/en_US/

adbeXXXX.msi  /qn /norestart disable_arm_service_install=1 eula_accept=yes disabledesktopshortcut=1 allusers=1

/qn silent install

/norestart  most likely not neccesary, but there just in case

disable_arm_service_install=1  new to the 10.1 package, this doesn't install ARM, which is nice.

eula_accept=yes   thought this fixed the eual thing, but users still get prompted first time they open reader up.

disabledesktopshortcut=1   preference

allusers=1   also i think this is unneccesary, just just in case.

Note

Download the installer from ftp://ftp.adobe.com/pub/adobe/reader/win/10.x/10.1.0/

Install using the command:

msiexec.exe /i ArchiveName.msi /qn /log install.txt DISABLE_ARM_SERVICE_INSTALL=1 EULA_ACCEPT=YES SUPPRESS_APP_LAUNCH=YES DISABLEDESKTOPSHORTCUT=1

DISABLE_ARM_SERVICE_INSTALL is new in version 10.1.X, this prevent the installation of update service.

Older version will be automatic replaced.
