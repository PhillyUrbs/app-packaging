Command Line
This is valid for V. 2.8.6 . It took me around 2 Days to figure this out, but know everything is working fine.

How To silent install:
1. Download easyMP Network Projection: https://ftp.epson.com/drivers/epson16189.exe

2. Extract the downloaded .exe (with 7-Zip for example) and you will get a second .exe, extract this one also and you will get 2 folders(InstData and Language)

3.  Open InstData and go into "license" Folder. From there copy the desired language into InstData and after that delete the license folder 
Warning: If you do not delete the license Folder you will end up with a License Agreement Pop-Up during Silent Installation)

4. Create setup.iss (InstData\setup.exe /r f1C:\temp\install.iss)  
Warning: Always run cmd as Admin or you don't have all needed entries in your install.iss File)

5. How To Silent-Install: InstData\setup.exe /s install.iss
Warning: install.iss has to be in the same folder as setup.exe, Full Path to the iss File won't work with /s (try instead /s f1<path>)

The Uninstall-Part is a bit more complex. First of all read the Inventory Records for EasyMP Network Projection then do the following:
1. Move to C:\Program Files (x86)\InstallShield Installation Information\ . There you will find a Folder for each installed InstallShield-Exe. Locate the folder for easy MP Network Projection (the files in the folder will be the exactly same as in instData when you run the installation).

2. Copy the following command and change the Path to be correct for you:
"C:\Program Files (x86)\InstallShield Installation Information\{4A515955-A3D4-4FE6-98C0-E7987FF3279A}\setup.exe" -runfromtemp -l0x0407 -removeonly "C:\Program Files (x86)\InstallShield Installation Information\{4A515955-A3D4-4FE6-98C0-E7987FF3279A}\setup.exe" -runfr  /r f1c:\temp\uninstall.iss

3. Run the above command and you will get a working uninstall.iss for silent uninstall.

4. How To Silent-Uninstall:
"C:\Program Files (x86)\InstallShield Installation Information\{4A515955-A3D4-4FE6-98C0-E7987FF3279A}\setup.exe" -runfromtemp -l0x0407 -removeonly "C:\Program Files (x86)\InstallShield Installation Information\{4A515955-A3D4-4FE6-98C0-E7987FF3279A}\setup.exe" -runfr /s uninstall.iss
Warning: This will cause an automatic reboot when uninstall is finished (no prompt etc)
Note: Again the uninstall.iss File has to be in the same Folder as setup.exe(try to use /f1 instead). 

I don't know why you can't do the Steps for Silent-Uninstall with the setup.exe used for installation. I've tried it, but ended up alyways with a -3 Error (Required Data not found in the uninstall.iss file).

[Method to change the mask image] The features and options of silent installation version
The features of the silent installation version:
The silent installation version does not display any messages while this software is installed to PC.
ü A message does not appear when the installing is completed.
ü The PC is rebooted without a message when user updates this software from previous version and PC requires a restart.
User can specify the massage and behavior of installation as installation option as follows.

Installation’s Options
User specifies the following options.
Format:
>EMP_NSCInstSilent.exe /xxxxx


Installation Options
Parameters Descriptions
/ep_reboot PC is rebooted when user wants to reboot after the installation is completed.
/ep_disp_end The completed message is displayed on a screen when the installation is completed.
/ep_disp_restart The message to reboot a PC is displayed on a screen when user updates the software from previous version.

Options of initial settings:
User can specify the initial setting by specifying the following values in “nscsettings.ini”. The “nscsettings,ini” is in “InstData” folder.

Initial settings
Section name Key name Key value and descriptions
[initialize] StartNICType [Features]
Specify the initial network adaptors that user uses when starting EasyMP Network Projection.

[Value]
0: Not specify the network adaptor automatically. (User specifies the network adaptor manually when user starts EasyMP Network Projection.
1: Specify the used network adaptor automatically.
2: Specify the wireless network adaptor automatically when user starts EasyMP Network Projection.
3: Specify the wired network adaptor automatically when user starts EasyMP Network Projection.

Default settings: 0


StartMode [Features]
Specify the start mode of EasyMP Network Projection

[Value]
0: Select the start mode when user starts EasyMP Network Projection
1: Specify Quick Connection mode when user starts EasyMP Network Projection
2: Specify Advanced Connection mode when user starts EasyMP Network Projection

Default: 0
StartSearchType [Features]
Specify the search mode when user starts EasyMP Network Projection.

[Values]
0:EasyMP Network Projection searches projectors by “Automatic Search”
1: Select the search mode when user starts EasyMP Network Projection.
2: EasyMP Network Projection searches projectors by what user used last time.

Default: 0
FullScreenMovie [Features]
Enable “full-screen video transfer” mode ON/OFF.

[Values]
0: Disable “full-screen video transfer”
1: Enable “full-screen video transfer”

Default: 1
InstallVirtualDisplay [Features]
Enable / Disable EPSON Virtual Display function.

[Values]
0: No install “Add/Remove EPSON Virtual Display” to PC.

Specify the number of installing EPSON Virtual Display .

0: No install EPSON Virtual Display
1: Install one EPSON Virtual Display to PC

Script
To suppress the prompt asking for firewall exceptions to be created, you must create Windows Firewall rules locally, not via Group Policy. The software does not correctly check for the presence of firewall rules created via GPO*.

I used the following PowerShell script on Windows 8.1 x64 to create inbound and outbound rules. Replace %ProgramFiles(x86)% with %ProgramFiles% if you are installing on 32-bit Windows. Once this script was run, EasyMP no longer prompted to create rules at startup.

New-NetFirewallRule -Direction Inbound -DisplayName "EasyMP Network Projection" -Program "%ProgramFiles(x86)%\EPSON Projector\EasyMP Network Projection V2\EMP_NSC.exe" -Action Allow -EdgeTraversalPolicy Allow
New-NetFirewallRule -Direction Outbound -DisplayName "EasyMP Network Projection" -Program "%ProgramFiles(x86)%\EPSON Projector\EasyMP Network Projection V2\EMP_NSC.exe" -Action Allow

* This is probably due to it only checking for rules under HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\SharedAccess\Parameters\FirewallPolicy\FirewallRules, whereas GPO firewall rules are defined in HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\WindowsFirewall\FirewallRules

Note
Some modification has been done in previous and it worked for me with batch file. No uninstall.iss require for uninstall.

Note : If other version or same version is already installed then setup will uninstall it first and reboot is must require before installation.
The uninstall command will not run silently and user interaction is require. So to avoid this situation and install latest version without reboot follow below steps

How To silent install:
1. Download easyMP Network Projection: https://ftp.epson.com/drivers/epson16189.exe

2. Extract the downloaded .exe (with 7-Zip for example) and you will get a second .exe, extract this one also and you will get 2 folders(InstData and Language)

3.  Open InstData and go into "license" Folder. From there copy the desired language into InstData and after that delete the license folder 
Warning: If you do not delete the license Folder you will end up with a License Agreement Pop-Up during Silent Installation)

4. Create setup.iss (InstData\setup.exe /r f1C:\temp\install.iss)  
Warning: Always run cmd as Admin or you don't have all needed entries in your install.iss File) Copy install.iss and paste into InstData folder

5. How To Silent-Install: \InstData\setup.exe -s -f1\InstData\install.iss
Warning: install.iss has to be in the same folder as setup.exe

The batch file will look like this: (Without reboot latest version will install )
if exist "%ProgramFiles(x86)%\EPSON Projector" rd /s /q "%ProgramFiles(x86)%\EPSON Projector"
if exist "%ProgramFiles%\EPSON Projector" rd /s /q "%ProgramFiles%\EPSON Projector"
if exist "%ProgramData%\SEIKO EPSON CORPORATION" rd /s /q "%ProgramData%\SEIKO EPSON CORPORATION"
if exist "%commonprogramfiles(x86)%EPSON Projector" rd /s /q "%commonprogramfiles(x86)%EPSON Projector"
if exist "%commonprogramfiles%EPSON Projector" rd /s /q "%commonprogramfiles%EPSON Projector"
if exist "%ProgramFiles(x86)%\InstallShield Installation Information\{4A515955-A3D4-4FE6-98C0-E7987FF3279A}" rd /s /q "%ProgramFiles(x86)%\InstallShield Installation Information\{4A515955-A3D4-4FE6-98C0-E7987FF3279A}"
if exist "%ProgramData%\Microsoft\Windows\Start Menu\Programs\EPSON Projector" rd /s /q "%ProgramData%\Microsoft\Windows\Start Menu\Programs\EPSON Projector"
reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\SEIKO EPSON CORPORATION" /f
reg delete "HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\SEIKO EPSON CORPORATION" /f

:: Start of Installation
::Installation path
....\InstData\setup.exe -s -f1....\InstData\install.iss

::END
