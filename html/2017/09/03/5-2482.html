Note
There is a LaunchCondition that forces you to use setup.exe. You can delete this from the LaunchCondition table.

1. Make a backup copy of your original IP52.msi
2. Open IP52.msi with Orca
3. Select the LaunchCondition table
4. Drop the row "Installed OR MM"
5. Save the file

After that you are able to create a MST with any tool you like to define the device you use.

Command Line
To deploy silently Intellipoint, you will need to use oemsetup and create an oemsetup.ini.

The syntax of oemsetup.ini is:
[Mouse]
Language=1036
DEVICE=IM
SHORTCUTS=0

The code for english is 1033
Device=
IM : Microsoft Intellimouse
IME : Microsoft Intellimouse Explorer
IME3 : Microsoft Intellimouse Explorer 3.x
IME4 : Microsoft Intellimouse Explorer 4.x
IMO : Microsoft Intellimouse Optical
IMW : Microsoft Intellimouse Web
MM : Microsoft Mouse
TE : Microsoft Trackball Explorer
TO : Microsoft Trackball Optical
WMO : Microsoft Whell Optical
WIME : Microsoft Wireless Intellimouse Explorer
WIME2 : Microsoft Wireless Intellimouse Explorer 2.x
WWM : Microsoft Wireless Whell Mouse
BOM : Microsoft Bnasic Optical Mouse
WOM : Microsoft Optical Mouse
WOM2 : Microsoft Optical Mouse 2.x

SHORTCUTS=0
0= no shortcut on the desktop
1= shortcut on the desktop

Command line:
cmd /c oemsetup.exe oemsetup.ini

Command Line
This is a silent install of the IntelliPoint 6.3 software.

(.\ is the root of the intellipoint install CD):

.\ipoint\setup\ip63.msi /qn REBOOT="ReallySuppress" MM=(mouse model)

Mouse Models can be found in the .\ipoint\setup\files folder in an XML file called ipointdevices.xml

However, although this installation runs properly it will throw errors when attempting to run the Control Panel applet if the prerequisites are not installed.

In order to get the software to install and launch successfully, you must ensure the proper prequisites are installed, as per the prereq folder included with the installation files.

Therefore, a working silent install of IntelliPoint 6.3 for the Wireless Notebook Presenter Mouse 8000 would look like this (substitue your own mouse model):

.\prereq\windowsinstaller3.1v2\x86\windowsinstaller-kb893803-v2-x86.exe /quiet /norestart
msiexec.exe /i .\prereq\watson\x86\dw20shared.msi /qn
msiexec.exe /i .\prereq\msxml\x86\msxml6.msi /qn
msiexec.exe /i .\ipoint\setup\ip63.msi /qn REBOOT="ReallySuppress" MM=39
WindowsXP-KB924941-v2-x86-ENU.exe /quiet /norestart

Windows Hotfix is a BlueTooth hotfix that you will be continually prompted for - thus, it is included in the installation (must be downloaded from Microsoft)

Note
Based on the tip from slowik above:

This applies to IP63.MSI

1. Make a backup copy of your original IP63.msi
2. Open IP63.msi with Orca
3. Choose New Transform
4. Select the LaunchCondition table
5. Drop the row "Installed OR MM"
6. Choose Generate Transform

You can now use your packaging tool to customise a transform by pointing it at the original vendor MSI with the original transform as a starting point.
This is almost the same as slowik suggested but leaves the original MSI unaltered.

Change the Property DSKSHORTCUT to 0 to eliminate the desktop shortcut
