Note
The easiest way i found of packaging this older application, included the following actions:

- Using Admin Studio i did a snapshot of the install.
- I included the little app, setACL in the install.
- I kept the Per User regsitry and application data otherwise i got errors.

Once packaged into an msi, i then created a custom action to give Everyone modify rights to all the corel folders.

I also added vb script to remove the Corel folder on uninstall as for some reason it wouldnt uninstall all the files.

FYI: TO get a custom action to run only on uninstall at a criteria : REMOVE="ALL"
