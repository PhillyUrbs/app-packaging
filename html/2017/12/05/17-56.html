Note

For Version 18:

Run the MSI with /qn /norestart

Suppress Update with this reg key:

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Update]

"Update{4DC8B4CA-1BDA-483E-B5FA-D3C12E15B62D}"=dword:00000000
