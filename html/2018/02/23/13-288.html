Command Line
I achieved silent installation and licensing of Dragon Naturally Speaking 13.0 Premium through an SCCM Task Sequence (run command) with updates disabled via the below command:

setup.exe /s /v" /quiet /Liwmo!e+ "%WINDIR%\Temp\DNSPremium13.0-Install.log" SERIALNUMBER=XXXXX-XXX-XXXX-XXXX-XX PRODUCTUPDATEFLAG=0
Please be aware that the product must still be activated on first launch, I am not sure if this can be achieved silently; especially considering the license I was using is a single-user license. I imagine for volume licensing scenarios there are other options.

Hope that helped someone!
