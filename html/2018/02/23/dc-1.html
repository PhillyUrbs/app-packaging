Command Line

After customizing my MST file using the Adobe Customization Wizzard DC, i used this command line in SCCM 2012 to deploy as an Application.

msiexec /i "AcroPro.msi" TRANSFORMS="myMSTfilename.mst" /q

With previous versions, i had issues installing with Microsoft Outlook open but with this version (DC), SCCM simply did a popup message saying a reboot is required to complete the installation. 


In my MST file i specified the following items:

  • Personalization Options - license key (also pressed the button "grant offline exception & disable registration")
  • Personalization Options - suppress EULA
  • Installation Options - make Acrobat default PDF viewer
  • Installation Options - remove previous versions
  • Installation Options - run silently (no interface)
  • Installation Options - suppress reboot
  • Online Services - Prompt users with open/save dialog and Disable Upsell

2. Run command: MsiExec.exe /Uninstall {AC76BA86-1033-FFFF-7760-0E0F06755100} /quiet
