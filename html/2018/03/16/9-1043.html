Note

For those who need to silently uninstall JAWS

 

/Type silent - Uninstalls JAWS.

  /Type silentRemoveShared - Uninstalls JAWS and removes all shared components.

be aware, that for uninstall the silent needs to be lowercase letters

Note
Heres the wise script I used to silently install Jaws 9.0 (I delete 2 shortcuts that we think users shouldn't have access to)

Also this does not take into account the activation

item: Check if File/Dir Exists
Pathname=%INST%\J9.0.2152.400-enu.exe
Flags=00000100
end
item: Execute Program
Pathname=%INST%\J9.0.2152.400-enu.exe
Command Line=/Type Silent
Flags=00000010
end
item: End Block
end
item: Get Environment Variable
Variable=ALL_USERS
Environment=ALLUSERSPROFILE
end
item: Check if File/Dir Exists
Pathname=%ALL_USERS%\Start Menu\Programs\JAWS 9.0\Tools\Remove Product Activation.lnk
Flags=01000100
end
item: Delete File
Pathname=%ALL_USERS%\Start Menu\Programs\JAWS 9.0\Tools\Remove Product Activation.lnk
end
item: End Block
end
item: Check if File/Dir Exists
Pathname=%ALL_USERS%\Start Menu\Programs\JAWS 9.0\Tools\Uninstall JAWS.lnk
Flags=01000100
end
item: Delete File
Pathname=%ALL_USERS%\Start Menu\Programs\JAWS 9.0\Tools\Uninstall JAWS.lnk
end
item: End Block
end

Heres The Uninstall Script
item: Get Environment Variable
Variable=PFILES
Environment=PROGRAMFILES
end
item: Check if File/Dir Exists
Pathname=%PFILES%\Freedom Scientific Installation Information\356DE2A8-01EB-464e-9C33-0EEA3F923000-810\UninstallJAWS.exe
Flags=00000100
end
item: Execute Program
Pathname=%PFILES%\Freedom Scientific Installation Information\356DE2A8-01EB-464e-9C33-0EEA3F923000-810\UninstallJAWS.exe
Command Line=/Type silent
Flags=00000010
end
item: End Block
end
