Command Line
First install CDBurnerXP with your settings:

cdbxp_setup_4.x /SAVEINF=default.cfg

After installation is done the settings are stored in your default.cfg. To install with these settings call:

cdbxp_setup_4.x /LOADING=default.cfg /SILENT

Note
To remove language pop-up on the first launch, add this record to registry:
[HKEY_CURRENT_USER\Software\Canneverbe Limited\CDBurnerXP]
"DefaultLanguage"="System.String:fr-FR"
en-US for english, sv-SE for swedish etc...

Command Line
I used the following command line to install silently, provide access to all users, associate with .iso files and not have a desktop shortcut.

cdbxp_setup_4.3.0.1991.exe /VERYSILENT /SP- /NORESTART /TYPE="full" /MERGETASKS="!desktopicon,isoassoc"

All of CDBurnerXP's setup info can be found at:
http://cdburnerxp.se/downloads/sourcecode/CDBurnerXP-Installer.iss

Explanations of the possible install/uninstall options can be found at:
http://unattended.sourceforge.net/InnoSetup_Switches_ExitCodes.html
