Note

TO INSTALL MSI SILENTLY & DISABLE AUTOUPDATES

1) Download MSI from: http://www.adobe.com/products/shockwaveplayer/shwv_distribution3.html

2) Edit MSI in Orca, go to Transform menu --> New Transform

3) Make the following changes:

Table: Property
Property: ISCHECKFORPRODUCTUPDATES
Value: 0

Change value from y to n for all of the following keys.

Table: Registry
Key: SOFTWARE\Adobe\Shockwave 12\AutoUpdate
     SOFTWARE\Adobe\Shockwave 12\collectstats
     SOFTWARE\Adobe\Shockwave 12\uicontrol\CollectStatistics
     SOFTWARE\Adobe\Shockwave 12\uicontrol\defaultautoupdate
     SOFTWARE\Adobe\Shockwave 12\uicontrol\defaultcollectstats
     SOFTWARE\Adobe\Shockwave 12\uicontrol\DoMandatoryUpdates
     SOFTWARE\Adobe\Shockwave 12\uicontrol\OfferOptionalUpdates
     SOFTWARE\Macromedia\Shockwave 10\AutoUpdate
     SOFTWARE\Macromedia\Shockwave 10\CollectStatistics
     SOFTWARE\Macromedia\Shockwave 10\uicontrol\CollectStatistics
     SOFTWARE\Macromedia\Shockwave 10\uicontrol\defaultautoupdate
     SOFTWARE\Macromedia\Shockwave 10\uicontrol\defaultcollectstats
     SOFTWARE\Macromedia\Shockwave 10\uicontrol\DoMandatoryUpdates
     SOFTWARE\Macromedia\Shockwave 10\uicontrol\OfferOptionalUpdates
Value: n

4) Save Transform file (Transform menu --> Generate Transform)

5) Install using the following command: msiexec /i INSTALLFILE.msi TRANSFORMS=TRANSFORMFILE.mst /qn
   (Replace INSTALLFILE and TRANSFORMFILE with the relevent file names)

Note
It is really important to start with the correct installer.
You must get 'sw_lic_full_installer.msi' which is downloaded as an MSI, not inside an EXE etc.
I spent ages messing with Shockwave_Installer_Full.exe but this contains a crappy little MSI that only installs a Merge Module for VC++ v9.
You get the MSI from the Shockwave Player Distribution Downloads page not the standard Adobe downloads page.
It is a dull white background page which contains amongst other things a link to 'Download MSI Installer'
You are only supposed to get to this site once you have got your Adobe licence, which is free.
I am not going to publish it here as Adobe threaten all sorts of nastiness if you do!

Note
This method works well.  I had actually only once before created a transform with Orca and the instructions were very easy to follow for the 1 property and 14 registry changes...which actual was 24, 2 for each default and +.  I wish I would have thought to sort the registry key column alphabetically first, that would have sped things up but I did afterwards and it was easier to double check that I didn't miss anything.  Also after deployment I checked the registry under

HKLM > SOFTWARE > Wow6432Node

and all the "n's" were in place.
