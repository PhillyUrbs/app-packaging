Note
The main installer for this app is an InstallShield EXE which can be driven by an ISS response file. However, the brain-dead packager at the 3rd-party used by this vendor omitted to silence the driver installation. When approached, Casella said, in essence, "Tough. We're too busy to bother with it." So, it fell to me to make it silent. Yes, I could have used AutoIt but there's no challenge in that!

Note that running the extracted MSI on its own fails to install the MySQL runtime or the scripts required to build the tables so forget that idea! Obviously, Mindteck found running it from a Custom Action too taxing.

During the installation:
- I copied the folder %TEMP%\{94991EC0-C537-47D4-A991-E9A1F3D96A1C} to a safe location, so that I had access to the EXEs, SQL scripts, etc. being used.
- I ran ProcMon so that I could capture the command lines being used.

Here's the resulting CMD. You can ignore entries for tools we use at the the client site (e.g. NOW). Hopefully, if you're reading this, you'll have the intelligence to map the EXE names mentioned with those that get extracted. If not, you should probably not be anywhere near work like this. Anyway...

SETLOCAL
set wrappername=Casella_Insight
set tools=C:\support\tools\0.Utils\
set logfile=C:\Support\logs\SCCM\%wrappername%_Wrapper.log
SET INSTALLOCATION=%~dp0
"%tools%now" -- Start of %wrappername% Install -- > %logfile%
:: The Casella tools write to this log file and expect it to be present.
set SQLLogFile=C:\Support\logs\SCCM\%wrappername%_MySQLSetup.log
"%tools%now" -- Create shell for MySQLSetup.LOG -- >> %SQLLogFile%
:Install_Drive
set exename=setup.exe
"%tools%now" -- Install %exename% -- >> %logfile%
"%INSTALLOCATION%app\Drive\%exename%" /S /NCRC
if NOT %errorlevel% == 0 goto :error
:Install_Insight
set msiname=Casella Insight.msi
"%tools%now" -- Install %msiname% -- >> %logfile%
msiexec.exe /i "%INSTALLOCATION%app\Insight\%msiname%" /qn ALLUSERS=1 REBOOT=ReallySuppress ROOTDRIVE=C:\ /lv "C:\Support\logs\SCCM\%wrappername%_%msiname%.log"

:Install_Driver_Ports
