Note
To get a copy of the MSI installer for VMWare Workstation 12:
Run VMware-workstation-full-12.X.X-XXXXXXX.exe

Just leave the installer running but don't click Next.

The MSI can be found in:
C:\Program Files (x86)\Common Files\VMware\InstallerCache\

Grab a copy, then exit the installer.
Setup Information:
Setup Type: Windows Installer (MSI)
Deployment Method Used: Windows Installer Command Line (No MST)
Deployment Difficulty: unspecified

Note
Some properties you can set:
EULAS_AGREED=1
DESKTOP_SHORTCUT=1
QUICKLAUNCH_SHORTCUT=1
AUTOSOFTWAREUPDATE=0
DATACOLLECTION=0
SERIALNUMBER=XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

You can add these to the property section of your MST file using a tool like Orca, or you can use them directly on the command line (with either the EXE or MSI).

Command Line
Use the following command line for a silent, unattended installation of Workstation 12 Pro that does not reboot at the end:
/s /nsr /v/qn EULAS_AGREED=1 AUTOSOFTWAREUPDATE=0 DATACOLLECTION=0 DESKTOP_SHORTCUT=0 ADDLOCAL=ALL DISABLE_AUTORUN=0 QUICKLAUNCH_SHORTCUT=0 REBOOT=ReallySuppress SERIALNUMBER="XXXXX-XXXXX-XXXXX-XXXXX-XXXXX"
Setup Information:
Setup Type: Windows Installer (Delivered as an EXE)
Deployment Method Used: Vendor Provided Command Line (switch driven)
Deployment Difficulty: Somewhat Easy
Platform(s): Windows
