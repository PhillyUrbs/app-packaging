Command Line
TO INSTALL MSI SILENTLY & DISABLE AUTOUPDATES in SCCM 2012
Tips for Foxit Reader 7.3.0.1228 :
If you are deploying Foxit Reader, you must sign up for the free enterprise version that comes as an MSI, known as Foxit Enterprise Reader, at http://www.foxitsoftware.com/pdf/reader/enterprise_reg.php. 
Alternatively you may be able to download Foxit Enterprise Reader from a  http://wpkg.org/Foxit_Reader
1. Download MSI file .
2. Edit the MSI-file in Orca :
3. Make the following changes:
    In Table Property 
    Poperty : AUTO_UPDATE 
    Value : 0 
4. Save msi file 
5. Install using the following command : 
msiexec.exe /i "%~dp0xxxxxxxxxxxSetup.msi" DESKTOP_SHORTCUT="0" MAKEDEFAULT="0" VIEWINBROWSER="0" LAUNCHCHECKDEFAULT="0" /passive /norestart /quiet'

I just found some of my notes... here are some reg keys to turn off some of those things we don't like,

 

[HKEY_CURRENT_USER\Software\Foxit Software\Foxit Reader 6.0\plugins\Updater]
"UpdateMode"="0"

[HKEY_CURRENT_USER\Software\Foxit Software\Foxit Reader 6.0\Preferences\General]
"bShowStartPage"="0"


"bShowAdvertisement"="0"

[HKEY_CURRENT_USER\Software\Foxit Software\Foxit Reader 6.0\Preferences\Registration]
"bShowRegisterDlg"="0"
