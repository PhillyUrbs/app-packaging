    118
    Note
    To install Corel Draw 11 without using setup.exe, make the
    following changes:

    - Ensure that "ISScript.msi" has already been installed
    - Add a property "ISSETUPDRIVEN" and set the value to "1"
    - Remove or comment out the "OnCheckSilentInstall" custom actin in the "InstallExecuteSequence"

    Source: Curtis Sawin WinInstaller MSI Mail List

    118
    Note
    1) Create an admin install of Corel.
    2) Apply SP2 to admin install.

    Then modify Corel msi in admin install:
    3) Add Property table value "ISSETUPDRIVEN" and set the value to 1.
    4) In the "Property" table set "ALLUSERS" to 2.
    5) Remove "OnCheckSilentInstall" from the "InstallExecuteSequence" table.
    6) Copy DR11.dta (from a PC with Corel installed and registered) into the admin install. File goes in \Program Files\Corel\Corel Graphics 11\Programs7) Ensure that the msi will install DR11.dta. I did this using WPS -> Windows Install Editor. Had to ensure that the instruction to remove the file on install is removed also.
    8) In the Components Table find "CommonPrograms"
    and add "DR11.DTA" to "KeyPath".
    9) In the File Table, add a row with the following Values:
    File:DR11.DTA
    Component:CommonPrograms
    FileName:DR11.DTA
    FileSize:104
    Attributes:8192
    Sequence:1834
    10) In the Directory tablefind "CorelDRAW11_Folder"
    and change "DefaultDir" to
    "Corel~1|Corel Graphics Suite 11".

    Now deploy using a GPO. Ensure that ISScript.msi is installed prior to CorelDraw.

    117
    Note
    Changing Corel Suite 11, so it can be installed by GPO.
    1. Generate an administrative Installation (setup /a) to
    generate an distribution point on a server.
    2. apply service packs 1 + 2.
    3. Go to http://www.palmOne.com/us/support/
    downloads/windesk41SE_legal.html and download the
    Palm Desktop Software 4.1! (We need a component out
    of this installation to update Corel!)
    4. Begin the installation of the Palm Desktop Software 4.1
    and when asked, give it place to store its Files. Wait
    for the Welcome Message and the first "Next >" Button.
    Now go to place you gave for storing and copy the
    file "Palm Desktop.msi" to your own location.
    5. Start Orca and load "Palm Desktop.msi"
    6. In Orca go to the Binary table and double click on
    [Binary Data] behind ISScriptBridge.dll, save it to
    ISScriptBridge.dll.
    7. Load Corel Graphics Suite 11.msi to Orca.
    8. In the Property table set ISSETUPDRIVEN to 1.
    9. In the Property table set ALLUSERS to 2.
    10. Go to the Binary table and double click on
    [Binary Data] behind ISScriptBridge.dll, load
    ISScriptBridge.dll from Palm Desktop.
    11. In the Binary table, delete the row which begins
    with "ISRT.dll" (this becomes obsolete with the new
    ISScriptBridge.dll).
    12. In the Menu Tranform, choose "New Transform".
    13. In the Directory table, search for "CorelDRAW11_Folder"
    and change "DefaultDir" to
    "Corel~1|Corel Graphics Suite 11".
    14. In the Components Table, search for "CommonPrograms"
    and add "DR11.DTA" to "KeyPath".
    15. In the File Table, add a row with the following Values:
    File:DR11.DTA
    Component:CommonPrograms
    FileName:DR11.DTA
    FileSize:104
    Attributes:8192
    Sequence:1834
    16. In the Property table, add the Item "InstallMode" with
    the Value "Complete".
    17. In the Property table, add the Item "INSTALLLEVEL" with
    the Value "3".
    18. In the Property table, add the Item "ProductID" with
    the Value "none".
    19. In the Feature table, search for "Draw11Suite", set
    Level to 3 and Attributes to 0.
    20. If you need to install something from the
    "Writing Tools", search for "WritingTools",
    set Level to 3 and Attributes to 2.
    21. Now you can switch of the features you don't need.
    On everything you need, let the Level Value be on 1,
    to switch one off, set the Level to 0.
    22. Save the Transform with "Menu Tranform",
    "Generate Transform...".
    23. So far, applying the changes with the transform,
    doesn't work on my Server. Therefore, i apply the
    transform to "Corel Graphics Suite 11.msi" and save
    the result with File¦Save transformed As... to a
    new file. That's it!
    24. To install the Package, make sure that IsScript.msi
    is already installed (via GPO or by Hand).

    This was tested with GPO-install with
    "advanced published and assigned" on a w2k-server.
    (Figuring out this process was supported with vital information from Curtis Sawin, thanks.).

    117
    Note
    Once you install Corel Draw 11 and enter the serial key.

    The Serial Key is stored in

    C:\Program Files\Corel\Corel Graphics 11\Programs

    copy the DR11.dta file and modify your Corel Graphics Suite 11.msi to include the above file that way you don''t have to enter the key on next install.
