    Note

    Deploy a whole configuration for the Firefox AddOn "UBlock Origin"


    Confgure youre example installation of UBlock Olrigin in Firefox.

    -       Back up the configuration to a textfile


    -       Copy the content of the backup file

    -       Got to http://raymondhill.net/ublock/adminSetting.html

    -       Paste the content to the first field

    -       Press the last -Button

    -       Copy the content of the field JSON-encoded settings to be used for adminSettings as a JSON string value

          

      Create a new file named "uBlock0@raymondhill.net.json"

          File-Content:

           

    {
     "name": "uBlock0@raymondhill.net",
     "description": "ignored",
     "type": "storage",
     "data": {
        "adminSettings":
      
            }
           
    }


       Paste the String to line 6 after "adminSettings":

          Save the file to "C:\Program Files\Mozilla Firefox\browser\extensions"

           Create the following Registry-Key:

        

    [HKEY_LOCAL_MACHINE\SOFTWARE\Mozilla\ManagedStorage\uBlock0@raymondhill.net]
    @="C:\\Program Files\\Mozilla Firefox\\browser\\extensions\\uBlock0@raymondhill.net.json"

            You can also use another path, UNC-path works fine for examle if you want a centralized file.


           Notice:

            This config overwrites it on UBlock Origin at every start of Firefox.
