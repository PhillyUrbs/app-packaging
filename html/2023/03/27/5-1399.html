    Installation:
    npp.5.7.Installer.exe /S

    Deinstallation:
    %ProgramFiles%\Notepad++\uninstall.exe /S

    No Updates:
    in the config.xml and the config.model.xml is the tag like:
    <GUIConfig name="noUpdate">no</GUIConfig>
    turn it to yes:
    <GUIConfig name="noUpdate">yes</GUIConfig>

    Both files are in %ProgramFiles%\Notepad++\

    9
    Note
    Version 5.9 out now!

    Silent Installation:
    --------------------
    npp.5.9.Installer.exe /S as usual.
    Note: this is case sensitive - be sure to use capital "S"!


    Disabling Updates:
    ------------------
    Just delete the "%ProgramFiles%\Notepad++\updater" folder.
    Note: there will be neither automatical update function nor a manual update option in the ? Help menu any more.

    Individual Localization:
    ------------------------
    1. per machine -> copy "%ProgramFiles%\Notepad++\localization\yourcountry.xml" to "%ProgramFiles%\Notepad++\nativeLang.xml"
    2. per user -> copy "%ProgramFiles%\Notepad++\localization\yourcountry.xml" to "%AppData%\Notepad++\nativeLang.xml" (user's profile)
    Note: If you chose the per machine way you avoid implementing Microsoft Active Setup or similar work ;-)
    Anyway, additional per user settings will allways be prefered.


    Icons & Shortcuts:
    ------------------
    No need to delete Start Menu shortcuts (e.g. uninstall.lnk, readme.txt) or Desktop icon any more - there aren't any.


    Silent Uninstall:
    -----------------
    "%ProgramFiles%\Notepad++\Uninstall.exe" /S as usual. Note: capital "S" again ;-)

    Note

    App: Notepad++

    Version: 6.1.5

    Operating System used for testing: Windows Server 2008 R2

    Installation: Virtualization

    Disabling Updates:

    ------------------------------------------------------------------------------------------------------------

    1. Just Uncheck the Auto Updater option during the installation of the application.This will suppress the update of notepad ++.
    2. Deleting the updater folder under the Program Files(x86)\Notepad++ would make no difference, since the auto update option was already removed during installation,so the gpup.exe responsible for update will not trigger
    3. Auto updates can also be suppressed by changing a section in config.model.xml under the %ProgramFiles%\Notepad++
      open this file and look for the following section
      <GUIConfig name="noUpdate">no</GUIConfig>
    4. change the above mentioned section to
      <GUIConfig name="noUpdate">yes</GUIConfig>

    Suppressing the updates of Plugins

    --------------------------------------------------------------------------

    1. Navigate to %AppData%\Notepad++\plugins\config folder
    2. Under this folder you would find the PluginManager.ini file
    3. Open this file. Its content would be like the following

    [Settings]
    LastCheck=1344415249
    ProxyPort=0
    Proxy=
    InstallLocation=0
    ProxyUsername=
    NotifyUpdates=1
    ShowUnstable=0

    Change the value of NotifyUpdates from 1 to 0
