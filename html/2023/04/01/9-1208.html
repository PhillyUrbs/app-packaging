    Note
    Problem: Sometimes the installation of MS Project does not work because of a dll conflict.

    Cause: MS Office 2002 Pro is installed and new webserver extentions are installed.

    Solution: uncheck the webserver extentions to be installed in the MS Project 2000 installation.

    Watch out when dealing with the 2000-era additional Office apps such as Project and Visio if you are working with Windows XP. Project attempted to update a protected file, I had to remove it from the package via transform before it would successfully install.
