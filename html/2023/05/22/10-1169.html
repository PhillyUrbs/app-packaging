    Note
    Dragon Naturally Speaking 10.0 comes with its own Visual C++ Runtime that must be installed before Dragon is installed. Unfortunately the VC Runtime that Nuance has included has a known bug that causes it to continually prompt for a reboot.

    Therefore the updated VC Runtime must be downloaded from here: DragonInstall.zip

    Then, for an unattended installation of Dragon Naturally Speaking 10.0, use the following commands:

    msiexec /qb! /i "Visual C++ Runtime for Dragon NaturallySpeaking.msi" SKIP_PENDINGREBOOT_CHECK=1

    Followed by

    msiexec /qb! /i "DragonNaturallySpeaking 10.msi" SKIP_PENDINGREBOOT_CHECK=1 SERIALNUMBER=xxxxx-xxx-xxxx-xxxx-xx

    Enter the 18 digit license key for the SERIALNUMBER property.

    See http://knowledgebase.nuance.com/view.asp?60VQ=KFHM for further details.
