    Note
    WinZip Command Line Support Add-On
    Use WinZip directly from the command prompt and scripts.
    This add-on requires WinZip 9.0.

    Silent install

    Copy wzcline -files to winzip folder:
    READMECL.TXT
    WZCL32.DLL
    WZCLINE.CNT
    WZCLINE.DLL
    WZCLINE.HLP
    WZCLSU.DLL
    WZCLUN.DLL
    WZUNZIP.EXE
    WZZIP.EXE

    Execute:
    %ProgramFiles%\Winzip\Winzip32.exe /autoinstall

    or:
    %ProgramFiles%\Winzip\Winzip32.exe /noqp /notip /autoinstall

    LANDesk 8 Install script:
