    Note

    Install ampagent on Mac manually via GUI

    Copy the macOS KACE SMA Management Agent installation file locally to the endpoint you want to install the KACE SMA Management Agent to and launch the installation file.

            a. Double-click ampagent-<version>.osx.all.dmg.

            b. Double-click AMPAgent.pkg.

            c. Follow the instructions in the installer.

    Be sure to enter the name of your KACE SMA

    Note

    Install ampagent on Mac manually via Terminal

    To set HOST with the Installer file (rename the msi installer file to include host name (FQDN) as follows (bolded)

            AMPAgent_kace_sma.example.com.pkg

            FYI: This will signal to the installation that kace_sma.example.com is the KACE SMA to connect to

    To set HOST with environment variable (has to be executed all on one line)

            sudo export KACE_SERVER=kace_sma.example.com installer -pkg '/Volumes/Quest_KACE/AMPAgent.pkg' -target /

    To set HOST with amp.conf (KACE SMA Management Agent configuration file)

            Ensure line in amp.conf for host entry is set

                    host=kace_sma.example.com
