    Note
    1.
    Install Lightroom on a system
    while installing copy the msi from %temp%\.... (its in the newest subfolder) to your source directory
    Start Lightroom and enter the serial number

    2.
    in your source folder create the following subfolder structures:
    \Windows\Profiles\Application Data\Adobe\Lightroom\Preferences
    and
    \All Users\Adobe\Lightroom

    3.
    Copy the .lrreg file from the system where you installed Lightroom under C:\ProgramData\Adobe\Lightroom to your source directory under \All Users\Adobe\Lightroom

    4.
    Create a file with the name "Lightroom 5 Preferences.agprefs" in your source directory under \Windows\Profiles\Application Data\Adobe\Lightroom\Preferences with the following content:

    prefs = {
        noAutomaticallyCheckUpdates = true,
    }


    5.
    Create a transform file for the msi with ORCA with the following entries (you may change the guids and names if you like)
    (you should be able to copy and paste this into ORCA, if not copy just value per value into the apropriate columns)

    Component Table:
    NoUpdate    {AAA06324-7163-46EF-A371-971C54B59B25}    AALPreferences    0        agprefs
    Registration    {5CE19C3D-18B9-4365-82DA-82EDDB435F70}    AUPALightroom    0        lrreg

    Directory Table:
    AALPreferences    AALightroom    Preferences
    AALightroom    AAdobe    Lightroom
    AAdobe    AppDataFolder    Adobe
    AUPALightroom    AUPAdobe    Lightroom
    AUPAdobe    ALLUSERSPROFILE    Adobe
    AppDataFolder    ProfilesFolder    .:Applic~1|Application Data
    ProfilesFolder    WindowsFolder    .:Profiles
    WindowsFolder    TARGETDIR    .:Windows

    FeatureComponents Table:
    LightroomBase    NoUpdate
    LightroomBase    Registration

    File Table:
    lrreg    Registration    Lightroom 5.0 Registration.lrreg    142        0    8192    5588
    agprefs    NoUpdate    Lightroom 5 Preferences.agprefs    52    0    0    8192    5589

    (check that the Sequence Numbers are greater than the biggest sequence number already in the msi)

    Media Table:
    2    5589    2        DISK2   

    (Check that the Diskid is higher than the highest already in the msi and check that the lastsequence number is equal to the highest sequence number you typed above for the file table entries)

    Property Table:
    REBOOT    ReallySuppress

    (you can set others too, if you like)

    Shortcut Table:
    alter the following entry:
    RunLightroom31321    ProgramMenuFolder    ADOBEL~1|Adobe Photoshop Lightroom 5 64-Bit    ShortcutStart_64    [INSTALLDIR]Lightroom.exe            0    NewShortcut31321_C2C2101F05384548B5AF39E0D3B3CB50.exe    0    1                   

    to

    RunLightroom31321    ProgramMenuFolder    ADOBEL~1|Adobe Photoshop Lightroom 5 64-Bit    Lightroom    LightroomBase            0    NewShortcut31321_C2C2101F05384548B5AF39E0D3B3CB50.exe    0    1                   

    (this will change the shortcut to advertised and so it will repair the product when the user clicks on the startmenu shortcut and within the repair the agprefs file will be copied to the appdata folder. So it won't work only for admin, it will work for every user then. the file is just copied if the file is not present.)
    (if you dont want the desktop shortcut delete the entry with Directory set to DesktopFolder and its gone)

    6.
    Save the transform file and install it with
    msiexec /i [msifile] TRANSFORM=[mstfile] /qb

    That's it.
    Have phun!
    Reto
