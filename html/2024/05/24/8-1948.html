    Hi All,

    using the command line switches to have a freeware Setup without the online updater or extra add-ons to purchase. No Shell extension or printer installed but added all the plugins for cloud services (OneDrive, SharePoint, GoogleDrive & Box).


    This is my install.cmd which i am deploying in a zip togehter with the "EditorV8.x64.msi".

    @echo off 
    msiexec /i EditorV8.x64.msi /qn ADDLOCAL="F_Viewer,F_Plugins,F_Plugin_SP,F_FileOpenPlugin,F_ReadOutLoudPlugin,F_SPPlugin,F_GDPlugin,F_DBPlugin,F_ODPlugin,F_BoxPlugin,F_VLangs,F_BrowserPlugins,F_NPPlugin,F_IEPlugin,F_VLangs" SET_AS_DEFAULT=1 VIEW_IN_BROWSERS=1 DESKTOP_SHORTCUTS=0 PROGRAMSMENU_SHORTCUTS=1 NOUPDATER=1

    Kind Regards

    Timo
