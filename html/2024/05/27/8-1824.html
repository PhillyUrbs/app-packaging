    Command Line
    MSI Installation:

    Unpack the EXE file to get the MSI and MST.

    Install silent (1031 is the german, choose your own one):
    Call MsiExec /I VMware Tools.msi TRANSFORMS=1031.mst REBOOT=REALLYSUPPRESS ARPSYSTEMCOMPONENT=1 /qn

    Repair silent:
    Call MsiExec /Fvomus VMware Tools.msi TRANSFORMS=1031.mst REBOOT=REALLYSUPPRESS ARPSYSTEMCOMPONENT=1 /qn

    Uninstall silent:
    Call MsiExec /X {FE2F6A2C-196E-4210-9C04-2B1BC21F07EF} REBOOT=REALLYSUPPRESS /qn /Li "%MSILogFile%"

    Note
    To active hardware acceleration, execute the following:

    echo off
    FOR /f "tokens=1-5* delims= " %%a in ( 'reg query HKLM\HARDWARE\DEVICEMAP\VIDEO /v \Device\Video0' ) do set VideoID=%%c
    reg add "HKLM\%VideoID:~18%" /v "Acceleration.Level" /t REG_DWORD /d "0" /f
