    Command Line

    For silent install:

    articulate-360.exe /quiet

    Install guide:

    https://cdn.articulate.com/assets/pdfs/articulate-360-deployment-guide.pdf

    This version of Articulate 360 allows admins to set 360 to update itself automatically, without requiring admin rights.  The provided instructions are unclear, however: it provides three scenarios and says that the first one is the default.  But that does NOT mean it will work without setting the reg keys!  

    To set Articulate to update itself or allow users to update itself:

    • HKLM\Software\Articulate\Common\Settings\DisableUpdates = false
    • HKLM\Software\Articulate\Common\Settings\DisableNonAdminInstalls = false 
    • HKLM\Software\Articulate\Common\Settings\DisableAutomaticUpdates = false 

    You can also set Articulate not to phone home with usage data:

    • HKLM\Software\Articulate\Common\Settings\PrivacyOptOut = true

    I tried putting it into App-V; it worked until I tried to set the update keys; those could be set, but on install appeared as "Type 0x4000001" which isn't a real thing.

    Note on auto-update: after Articulate 360 has been installed for a while, it appears that it will automatically update itself, as noted with a test machine that was left overnight.  If launched immediately after install, it will notify the user that an upgrade is necessary and wait for the user to press the Update button.  Presumably it would update itself eventually if left alone.

    If you're using SCCM, beware using the Product ID (GUID) for the detection rule.  That appears to change with each update, so you'll have computers reverting to the packaged version every night.  I used %ProgramFiles(x86)%\Articulate\360\Desktop Application\Articulate 360 Desktop App.exe GT or EQ 1.48.24116.0

    The app will upgrade itself: uninstalling the old version first is not necessary.

    Silent Uninstall:

    • 'C:\ProgramData\Package Cache\{b3d0ab58-3e17-40bc-a6bb-32d13da83d70}\Articulate.360.Bundle.exe' /quiet /uninstall

    Note that the uninstall does not remove 'HKLM:\Software\Articulate\'
