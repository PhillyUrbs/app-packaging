    Note

    Make a file named quicktimeinstaller.ini with these lines:

    [QTSETUP]

    32Bit=FALSE

    16BIT=FALSE

    NO_DIALOGS=TRUE

    SHOW_SAMPLE=FALSE

    SHOW_README=FALSE

    SHOW_PROGRAMFOLDER=FALSE

    REGISTRATION_DIALOG=FALSE

    LICENSE_OPTION=2

    SUPPRESS_SPEED_DIALOG=TRUE

    Put it in de same directory of your QuickTime files. Run the quicktime.exe without parameters.

    Note
    Here is a copy of a INI file which I worked on today which gets alot closer to a silent install the one issue I still have is I want to turn off the Check Update feature which at the moment the only way I have found to do this is with a SCRIPTIT script.

    [QTSETUP]
    32Bit=FALSE
    16BIT=FALSE
    NO_DIALOGS=TRUE
    SHOW_SAMPLE=FALSE
    SHOW_README=FALSE
    SHOW_PROGRAMFOLDER=FALSE
    REGISTRATION_DIALOG=FALSE
    LICENSE_OPTION=2
    SUPPRESS_SPEED_DIALOG=TRUE
    INSTALL_QTJAVA=TRUE
    INSTALL_QD3D=TRUE
    INSTALL_QTINFO=TRUE

    [QTPrefs]
    ConnectionSpeed=Intranet
    CSMultipleStreams=0

    The last two settings set the internet speed to Intranet/Lan and multiple streams .
    Can't find the setting for the Update Check though :(

    Here is my scriptit script to turn that off.

    [SCRIPT]
    Run=C:\Winnt\System32\control.exe
    Control Panel=Q#{ENTER}
    QuickTime Settings=#{TAB}#{DOWN 11}#{TAB 4}#{SPACE}#~winclose
    Control Panel=~winclose

    Hope this helps and if any finds the Other settings can you please post them.

    117
    Note
    Most of the configuration for QuickTime, including the automatic updates, is stored in QuickTime.qtp. On Windows 2000 this is found in:

    C:\Documents and Settings\All Users\Application Data\QuickTime\QuickTime.qtp

    You could create a QuickTime.qtp file with the settings you want and replace the default one with that. You can't change individual settings this way though.
