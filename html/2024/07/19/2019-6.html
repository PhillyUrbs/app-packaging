    Note

    Deployment is actually almost the same like Version 15. But you can now move Databases from an Backup silently.

    If you encounter

    • Error 117 your ini-File for Silent install is not found.
    • Error 118 you have to restart the Target-Machine before installation.
    • Error 223 there are errors in you're ini-File. Also make sure you're ini-File is saved in UTF8 without BOM.
