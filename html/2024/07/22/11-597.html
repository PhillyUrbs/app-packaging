    3
    Note
    We created a customisation using command...

    Setup.exe --record=1

    Changed the location of the start menu items by adding the following line to the .xml file created by the toolkit.

    <Property name="StartMenuSubFolder">C:\Documents and Settings\All Users\Start Menu\Programs\Adobe Photoshop CS4</Property>
    This worked for all shortcuts except the "Adobe Media Player", which we decided not to deploy to the machine anyway.
