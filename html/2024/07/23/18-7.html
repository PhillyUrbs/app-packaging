    Command Line
    setup.exe /t /qb /Language en-US /c ACADLT: INSTALLDIR="%ProgramFiles%\Autodesk\AutoCAD LT 2011\" ACADSERIALPREFIX=XXX ACADSERIALNUMBER=XXXXXXXX ADLM_PRODKEY="XXXXX" ACADFIRSTNAME="XXX" ACADLASTNAME="XXX" ACADORGANIZATION="XXX" InstallLevel=3 ALLUSERS=1 ADSK_DESKTOPSHORTCUT_1=0

    InstallLevel=5 - for full installation
    InstallLevel=3 - for typical installation
    ADSK_DESKTOPSHORTCUT_1=0 - to remove shortcut from desktop

    Command Line
    You have to make a admin install with the Repartition Creation tool on the AutoCAD DVD installation kit. When it's finish you have access to %Source_Path%\AdminImage and %Source_Path%\Tool folder.

    Command to install, uninstall and check the install :

    Install = '"%Source_Path%\AdminImage\Setup.exe" /qb /I %Source_Path%\AdminImage\AutoCADLT2011.ini /language en-US'

    Uninstall = '"C:\Program Files\Autodesk\AutoCAD LT 2011\Setup\Setup.exe" /QB /W /T /P {5783F2D7-9009-040C-0002-0060B0CE6BBA} /R ACADLT /language en-US'

    Verify = '"C:\Program Files\Autodesk\AutoCAD LT 2011\Setup\Setup.exe" /QB /W /T /P {5783F2D7-9009-040C-0002-0060B0CE6BBA} /M ACADLT /language en-US'

    RSVP : For french version, replace en-US by fr-FR... other language you have to verify your language code.
