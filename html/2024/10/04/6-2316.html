    Note
    This will with any of the NetBean packs.

    I will use c:\netbean as an example directory you can replace this with what ever you wish.

    1) Download the installer from their website.

    2) Place in a local folder for example c:\netbean

    3) Call the installer from the run command as follows

    "c:\netbean\netbean-6.0-windows.exe" -- record "C:\netbean\state.xml"

    This will run through an install once finnished it will record the configuration to state.xml which is similar to an answer file of mst.

    4) You can now perform a Silent install with the installer and the file you have created by using the following command.

    "c:\netbean\netbean-6.0-windows.exe" -- silent --state "C:\netbean\state.xml"

    5)Now that you have installed the product as apart of task 3 you can record and uninstall. Goto c:\program files\NetBeans and copy the Uninstall.exe to c:\NetBeans

    6)from the run command call the uninstaller as follows
    c:\NetBeans\uninstall.exe --record c:\netbeans\stateremove.xml

    7)This will uninstall netbeans from your machine and create the file stateremove.xml.
    You can now silently remove netbeans with the following command.

    "c:\netbeans\uninstall.exe" --silent --state "C:\netbeans\stateremove.xml"

    You can now copy the folder c:\Netbeans to your network drive and change the quiet install and uninstall commands to the relevant paths.

    Enjoy

    Note
    I've just built 6.7 in the same manner. The post above is incorrect either the issue has been resolved or the wrong command line was used. The correct ones are as above

    "c:\netbeans\uninstall.exe" --silent --state "C:\netbeans\stateremove.xml"
    "c:\netbean\netbean-6.0-windows.exe" --silent --state "C:\netbean\state.xml"
    "c:\netbean\netbean-6.0-windows.exe" --record "C:\netbean\state.xml"l

    Only issue I had was that the installer unpacks itself to your mydocuments\.nbi which was an issue as it pushed my account over its space restriction and error'd. You need at least 250mb spare. If upping space is an issue for some reason you can repoint the logs and temps with the following command.

    netbeans-6.7.1-ml-windows.exe --userdir "c:\documents and settings\dangay\localsettings\temp\.nbi"

    If you have any issues netbeans tech support are really quick to respond and very helpful!

    See links for where to go for extra info.
