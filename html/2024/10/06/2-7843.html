    Note
    The new installer comes with everything packed in one install and they don't appear to have an admin option.

    So download the installer, click next next next until it starts unpacking the apps. It will do this to c:\docs&Setting\username\local settings\temp\Smart Technologies\install rapper\temp\i

    Copy the Smart Response folder to somewhere safe.

    By default this installer will install SmartProductUpdate. This is contained in the SPU feature. So open in your favorite editor and delete this feature. Updater no more.

    Once done you can transform the install or alternatively edit the current msi and add/edit ISX_SERIALNUM in property table to include you serial number.

    Smart Response, Done!
