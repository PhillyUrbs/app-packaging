    Command Line
    Edit the setup.ini that is included in the unzipped file directory. Add your msi command lines (including the serial number property) to the [Startup] section. Here is an example:

    CmdLine=/passive /lv c:\logdir\logfilename.log SERIALNUMBER=xxxx-xxxx-xxxx-xxxx-xxxx-xxxx AgreeToLicense=yes REBOOT=No

    By passing the serial number in the command line, you do not have to capture and install the COMMONFILESFOLDER\Adobe\Adobe PCD\cache\cache.db file (as explained in the "Notes" section below).

    Note
    For Captivate 4, rename the registration.dll which will suppress the "Registration Your Software" window when the application is launched.

    REM Disables registration prompts
    RENAME "C:\Program Files\Adobe\Adobe Captivate 4\registration.dll" registration.dl_

    Creating the DWORD registry key "Enterprise" and setting the value to one (1) does not appear to make a difference for Adobe Captivate 4 as it did in Captivate 3.

    -6
    Note
    Execute Adobe_Captivate_3_WWE.exe to extract msi and associated data files.

    Run an admin install on the captivate_installer.msi to reveal the installation files.

    Licensing and Registration Dialog:
    -install Captivate 3 and start the application.
    -start whatever setup capture you use before you enter the license key. After the initial snapshot, enter your license key and select "don't register' at the registration dialog. Continue with the initial start of Captivate.
    -restart your setup capture and execute the second snapshot of your machine.

    The snapshot will note the following files...include them in your installation to avoid the display of the license key dialog and the registration dialog.

    COMMONAPPDATAFOLDER\FLEXnet\adobe_xxxxxxxx_tsf.data
    COMMONFILESFOLDER\Adobe\Adobe PCD\cache\cache.db
