    Note
    If doing a per-machine deployment, you will also need to add the same registry keys to HKCU. The application ignored the HKLM keys for subsequent users who login.
    HKCU\Software\Audotdesk\Design Review\Settings\AutoUpdate
    DWORD:InitialCheck = 1
    DWORD:DoNotShowAgain = 1

    To get the package to self repair, and hence populate the HKCU data, you will need to make the shortcut advertised.

    Integrate Design Review Service Pack 1 into the original Design Review 2008 Installation
    ********************************

    You will need to do an administrative install first:

    Msiexec -a [path to .msi file]
    You will be asked for a location to put the admin installer.

    Then patch the administrative image using the command-line:
    msiexec -a [path to administrative image .msi file] -p patch.msp

    For example, the msi is in c:\temp\setupdesignreview.msi, type the command:

    msiexec -a c:\temp\setupdesignreview.msi

    Choose the folder c:\temp\adradmininstall
    Then, type
    Msiexec Â–a c:\temp\adradmininstall\setupdesignreview.msi Â–p c:\temp\autodeskdesignreview2008_sp1.msp

    Now you have a patched (SP1) and uncompressed installation of Design Review 2008.

    Command Line
    The instructions for a silent install of Design Review 2008 are exactly as those posted for Design Review 2007.

    How do you get the .MSI file out of the .EXE file that you download? Run the EXE, then when you get to the first install dialog, stop and search your %temp% directory and you will find a file named SetupDesignReview.msi

    To review, here is the string to use in your BAT or script file.

    MSIEXEC /i "\\server\share\SetupDesignReview.msi" /l*v %temp%\DesignReview.log /qb!

    The above information & much much more can be found here at the "CAD Panacea" http://rkmcswain.blogspot.com/

    117
    Note
    To disable the automatic updates, add the 2 following registry keys via your MST, directly on the MSI or other automated methods.

    HKLM\Software\Audotdesk\Design Review\Settings\AutoUpdate
    DWORD:InitialCheck = 1
    DWORD:DoNotShowAgain = 1

    This is meant for design review, not DWG true view.

    Note
    Only the German version of DesignReview 2008 applies here...

    If SP1 will be automated installed, it breaks the installation with following error message off: Mistakes 1334. The file "DesignReview. chm" cannot be installed because the file in the Cabinet-file "PCW_CAB_Family00 cannot be found". This can refer to a network mistake, a mistake in the reading of the CD-ROMS or on a problem with this package. Previously yet the PrintPatch installs, tests with administrative images, functions also not.

    SOLUTION: Mistake in the SP1-Patch, wait for SP2.
