    Note
    First extract the installer MSI file using the /a switch

    In the resulting admin directory locate the install MSI and open with your favorite MSI editor (I used ORCA). Either edit the MSI directly or create a transform (MST).
    Browse to the PROPERTY table and add the following rows:

    Property=COMPANYNAME
    Value=your company name (If left blank this will default to the company name the host box is registered to)

    Property=USERNAME
    Value=the desired username (If left blank this will default the the computer name of the host box)

    You can make other changes here as well like setting the Accept Property to Yes to automatically accept the EULA.

    Now save the changes and run the install from the admin directory. Once the install completes launch the application and enter your license key info when prompted. Now close the app and browse to the install directory (usually %PROGRAMFILES%\Sparx Systems\EA) and locate the key.dat file. Copy this file to your Admin install folder under Program Files\Sparx Systems\EA, overwriting the existing file.

    DONE! You can run the install from your Admin folder MSI with the /qn and /norestart switches. The license key will install automatically for all users.

    Uninstall string is:
    MsiExec.exe /X{CC98E8B3-FAAA-4D09-A813-A44C9FA1A3EE} /qn /norestart
