    Here are some more functions for setup.ini:
    DISABLEEMSFEATURES=0
    The DISABLEEMSFEATURES option is used to disable the Photoshop.com membership features. Enter a value of 1 (DISABLEEMSFEATURES=1) to turn off the Photoshop.com membership features.

    DISABLEOLSFEATURES=0
    The DISABLEOLSFEATURES option is used to disable the retail online services and to display available educational online services. Enter a value of 1 (DISABLEOLSFEATURES=1) to disable retail online services and to display available educational online services.

    Note
    When you first run up Photoshop Elements 8 it prompts you for a Country/Region.To disable this set the following reg key(I modified 1033.mst) to do so:

    HKLM\Software\Adobe\Photoshop Elements\8.0\Country

    Set the value to your choice of country.

    Note
    There are a few options. All involve modifying the setup.ini.

    Silent install for English:
    EnableLangDlg=NO
    CommandLine0=msiexec /i "Adobe Photoshop Elements 8.0.msi" TRANSFORMS=1033.mst ELEMENTS_EN_US=1 ORGANIZER_EN_US=1 SERIALNUMBER=1234-5678-9000-0000-0000-0000 /qn

    Install with Langauge prompt:
    EnableLangDlg=YES
    *Then modify each CommandLine# with the appropriate MSI switches to perform as desired.

    2
    Note
    For deployment via group policy:

    As per the additional notes section of this link http://kb2.adobe.com/cps/520/cpsid_52063.html

    1. Extract all the files to a network share
    2. Edit the .msi with Orca to add the following properties:
    EULADelay = 0
    Registration = Suppress
    Updates = Suppress
    SERIALNUMBER=volume-license-serial-number
    3. Add to group policy SW install with transform for your language e.g. 1033.mst for English

    2
    Note
    If you don't want the desktop shortcut add this to the command line:
    NODESKTOPICON=1

    If you want to move the shortcut on the start menu, add this to the command line:
    SHORTCUTLOCATION=Adobe Photoshop Elements
    to put it in Start menu\Programs\Adobe Photoshop Elements

    You must add that to the uninstall command also to delete the shortcut.

    You can also change the properties in the mst file for your language instead.

    Note
    problem1:
    ---------
    Online Services erorr shown appears when launching organizer can be disabled using below method

    online services could not be initialized. Please reinstall. Press OK to continue without online

    try copying the ols_config.xml file from the your installer disk into the below location:

    C:\Program Files\Adobe\Elements Organizer 8.0\Assets folder.

    If an ols_config file already exists, then too replace it with the one from the installation disk/folder.

    problem2:
    ---------

    Disable Adobe Partner services and when shortcut is launched organizer elements organizer dialog pops up.The following files needs to be at this location


    %appdata%\Adobe\Online Services\Photoshop Elements 8.0.0\clients\Photoshop Elements

    files need to add: notifications.dat and preferences.dat

    %appdata%\Adobe\Online Services\Photoshop Elements 8.0.0\cache

    files need to add: cache.dat

    %appdata%\Adobe\Elements Organizer\8.0\Organizer

    files needs to add: psa.prf,APDpreferences.xml,status.dat,customevents.dat

    problem3:
    ---------

    The Adobe Photo Downloader is installed by default, and has an icon which appears in the system tray. Basically, all this application does is takes over the import of graphics files to your computer, whether Elements is running or not.

    To disable the Adobe Photo Downloader, set the following registry key value:

    HKLM\SOFTWARE\Adobe\Elements Organizer\8.0\Organizer

    "AutoLaunchOnDeviceConnect"=dword00000001
