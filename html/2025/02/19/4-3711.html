    Note
    We have found an issue with the Adobe Design Suite Premium CS4 and IBM Lotus Notes 8.5.

    When Lotus notes is installed on a machine and you install CS4 Premium, the install fails to successfully install Adobe Acrobat Professional. If CS4 Premium is installed first and then Lotus Notes is installed - all is good.
    this was not an issue with Lotus Notes 6.5, we can only summise that the new eclipse/java sections in Lotus 8.5 cause the issue.
