    Command Line
    Copy the installer to a network share.
    "\\networkshare\xpsoft\Gerald Combs\Wireshark 0.99.6a"

    Deploy with the following code.
    "\\networkshare\XPSoft\Gerald Combs\Wire Shark 0.99.6a\wireshark-setup-0.99.6a.exe" /S /desktopicon=no /quicklaunchicon=no

    NOTE - WinPcap will not install with this type of installation. WinPcap no longer supports silent switches, so you will need to repackage this app - look up WinPcap and check out what joedown has to say about repackaging.

    http://www.appdeploy.com/packages/detail.asp?id=1194

    • One small suggesstion here, Do not capture more than one applicaiton and create single msi for them..

      Here, WireShark and WinPCap both are different application..

      You are not supposed to capture them as single msi.. because WinPCAP will be pre-req for other application as well..

      * If a user wants to install some other application which needs WinPCAP as pre-req then you have to again package WinPCAP seperately...

      * You can not give him WiseShark+WinPCAP as pre-req since you have pacakged them as single package. For him WiseShark is unnecessary..

      * Later, if you are packaging WinPCAP seperately and installing it on user machine as a pre-req for some other application.. now if the user wants WiseShark then again you have to uninstall the WinPCAP from his machine and install WiseShark+WinPCAP..

      etc.. etc... - jagadeish 12 years ago
