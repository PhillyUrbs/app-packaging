    3
    Command Line
    I've been deploying Photoshop Elements 9 and Premiere Elements 9 together and found after the install that users were being prompted to select their country when starting PRE9 for the first time. I made the following change to my install command that seems to have worked for both packages...

    \\your.server.name\yoursharename\setup.exe /UL1033 /V"SERIALNUMBER=1234-5678-9012-3456-7890-1234 COUNTRY=242"

    This Country code is for UK. You can find the full list of Country codes in the ComboBox table inside the MSI package. The list of Country codes is DIFFERENT between the PSE9 package and the PRE9 package (although the UK is 242 in both), so check for your locale. This isn't documented anywhere else that I can find, so your mileage may vary.

    3
    Note
    I really had trouble using the setup.exe command which was recommended from Adobe. Somehow the setup was crashing on Windows 7 32-Bit computers, so I decided to look into the MSI Installation and got it fully working using following:

    1. User MSIEXEC to install "Adobe Photoshop Elements 9.msi" with your desired parameters:
    ELEMENTS_DE_DE=1 ORGANIZER_DE_DE=1 SERIALNUMBER=XXXX-XXXX-XXXX-XXXX-XXXX-XXX DISABLEEMSFEATURES=1 DISABLEOLSFEATURES=1 WATCHSERVICE=0 NOT_STANDALONE=1
    Eventually you have to change "ELEMENTS_DE_DE" and "ORGANIZER_DE_DE" to fit your language.
    Also select the MST that fits to your language.

    2. User MSIEXEC to install "ElementsOrganizer\Elements 9 Organizer.msi"
    Parameters: NOT_STANDALONE=1
    Transform: Your Language Transform from within ElementsOrganizer Folder.

    3. Use AAMEE ( http://blogs.adobe.com/oobe/2010/06/adobe_application_manager_ente.html ) and create a new package.
    Select the "ElementsSTIInstaller" Folder as Setup Source Directory. This will create an MSI Installater for all addional components like Camera Raw and Extended Script.
    Just install the new MSI with /qn switch.

    4. Install Adobe Air Player with silent switches (-silent -eulaAccept)

    5. Now we are just missing AdobeHelp. Check the "exceptions" Folder within the Destionation Folder you selected in Step 3.
    You can Install AdobeHelp from there using:
    AIRInstallerRunner.exe -airRuntime AdobeAIRInstaller.exe -silent -eulaAccepted -programMenu -airAppPath AdobeHelp.air
    (or see ExceptionInfo.txt)

    Hope that helps.

    -9
    Command Line
    It's possible to install the MSI directly by using NOT_STANDALONE=1. Complete command line would resemble something like this:

    msiexec /i "Adobe Photoshop Elements 9.msi" TRANSFORMS=1033.mst ELEMENTS_EN_US=1 ORGANIZER_EN_US=1 SERIALNUMBER=XXXX-XXXX-XXXX-XXXX-XXXX-XXXX DISABLEEMSFEATURES=0 DISABLEOLSFEATURES=0 WATCHSERVICE=0 NOT_STANDALONE=1 /qb


    [EDIT]

    When bypassing the setup.exe something does go wrong afterall. Somehow the Organizer isn't correctly installed (this is a seperate MSI located in the ElementsOrganizer subfolder of your installation source). Install the Organizer with the following command line:

    msiexec /i "Elements 9 Organizer.msi" TRANSFORMS=1043.mst NOT_STANDALONE=1 /qb
