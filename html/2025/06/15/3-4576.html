    Note

    Silent install Command:

    wireshark-win64-3.*.*.exe /S /desktopicon=no.


    To disable updates. 


    Goto Edit-> Preferences -> Advanced -> Type gui.update.enabled and double click on it to change the value to False.



    Which in turn saves the settings to %APPDATA%/Wireshark/preferences

    Copy the preferences file to the install directory which disables automatic updates.
